# 工作本质是自我营销

📅 [[2022-04-14]] 08:49:24

工作需要主动，提供价值，交流本就是一种自我营销与展示

---

## 🧠 深度解析

**核心洞察**：
这是一个关于「职场本质」的深刻认知。工作不只是做事，更是「自我营销」。你说的「交流本就是一种自我营销与展示」，点破了职场的底层逻辑：**你的价值，取决于别人对你的认知**。

**底层逻辑**：
- 主动 = 被看见 = 有机会
- 被动 = 被忽略 = 没机会
- 交流 = 展示价值 = 建立认知
- 不交流 = 价值归零 = 没人知道你

**实践方法**：
1. **主动汇报**：每周主动向领导汇报一次工作进展
2. **价值外化**：把做的事说出去，不要默默无闻
3. **交流策略**：每次交流前想清楚「我要展示什么价值」
4. **建立标签**：让别人提到某个领域就想到你

**内容创作应用**：
- 📱 短视频选题：「为什么默默做事的人，永远升不了职？」
- 📝 长文选题：「职场真相：你的价值，取决于别人对你的认知」
- 💬 金句卡片：「交流本就是一种自我营销与展示」

**爆点话题**：
- 🔥 「职场潜规则：不会展示的人，永远被埋没」
- 🔥 「为什么你做了很多，却没人知道？」

# 强者模式与绝对信念

📅 [[2022-04-15]] 13:48:52

---

## 🧠 深度解析

**核心洞察**：
这是一个关于「信念与行动」的深刻认知。核心观点：**强者因为相信所以去做，普通人因为看见了才相信**。这揭示了两种人生模式：活在未来 vs 活在当下。

**底层逻辑**：
- 相信 → 行动 → 看见结果（强者模式）
- 看见结果 → 相信 → 行动（普通人模式）
- 活在未来 = 从未来反推现在的行为
- 绝对信念 = 让未知变成相对确定

**为什么需要「绝对正确」？**
- 不确定 = 犹豫 = 不行动
- 绝对正确 = 坚信不疑 = 全力以赴
- 原则 = 判断标准 = 减少决策成本

**实践方法**：
1. **建立原则清单**：列出你坚信不疑的原则
2. **从未来反推**：想象5年后的自己，反推现在该做什么
3. **先相信再行动**：不要等看见结果才相信
4. **减少犹豫**：用原则代替纠结

**内容创作应用**：
- 📱 短视频选题：「为什么强者先相信，普通人先看见？」
- 📝 长文选题：「活在未来：从未来反推现在的行为」
- 💬 金句卡片：「因为相信，所以看见」

**爆点话题**：
- 🔥 「你是在相信未来，还是在等待未来？」
- 🔥 「原则的力量：减少90%的决策成本」

2. **扩展行动边界**：尝试、执行、复盘、迭代

# 钓鱼哲学与选择大于努力

📅 [[2022-04-16]] 16:12:54

钓鱼的第一原则是在有鱼的地方钓鱼活着的第一条原则就是不去会让自己死掉的地方。去合适的位置，找合适的时间，做合适的选择

---

## 🧠 深度解析

**核心洞察**：
这是一个关于「选择大于努力」的经典隐喻。钓鱼的第一原则不是「钓技多好」，而是「在有鱼的地方钓鱼」。人生的道理一样：**选择正确的位置，比努力更重要**。

**底层逻辑**：
- 有鱼的地方 = 有机会的地方
- 没鱼的地方 = 再努力也没用
- 不去会死的地方 = 风险控制
- 合适的位置 + 合适的时间 = 成功的一半

**三个核心原则**：
1. **位置原则**：在有鱼的地方钓鱼（选择大于努力）
2. **生存原则**：不去会让自己死掉的地方（风险控制）
3. **时机原则**：合适的位置 + 合适的时间（时机很重要）

**实践方法**：
1. **评估你的「鱼塘」**：你现在的地方，有鱼吗？
2. **换鱼塘**：如果没鱼，换一个地方
3. **风险清单**：列出会让你「死掉」的事，坚决不做
4. **时机判断**：现在是不是合适的时间？

**内容创作应用**：
- 📱 短视频选题：「为什么努力的人不一定成功？」
- 📝 长文选题：「钓鱼哲学：选择大于努力的三个原则」
- 💬 金句卡片：「在有鱼的地方钓鱼，比钓技重要100倍」

**爆点话题**：
- 🔥 「你的鱼塘，有鱼吗？」
- 🔥 「为什么有些人再努力也没用？」

所谓投资就是用现在资源换取未来的资源。所谓活在未来就是用未来的眼光来督促现在的行为，从而实现未来的某个更加确定的结果。活在未来是用现在有的资源换取未来的资源。因为有了活在未来，所以能够更加的合理的配置现有的资源。#李笑来 #活在未来#逻辑

---

### 🧠 深度解析

**核心洞察**：
这是一个关于「投资与活在未来」的精确定义。核心观点：**投资 = 用现在资源换未来资源**。活在未来 = 用未来眼光配置现在资源。

**底层逻辑**：
- 投资 = 现在资源 → 未来资源
- 活在未来 = 未来眼光 → 现在行为

- 确定性 = 通过行动让未来更确定

3. **用未来眼光**：问「5年后，这个资源会变成什么？」

---

### 🧠 深度解析

**核心洞察**：
这是一个关于「沉没成本」的营销案例。同样的优惠，提前赠送2个印花比从零开始效果好很多。为什么？**因为人不想「浪费」已经拥有的东西**。

**底层逻辑**：
- 从零开始 = 需要积累8个 = 感觉遥远
- 提前赠送2个 = 只需要积累6个 = 感觉接近
- 沉没成本 = 不想浪费已有的2个 = 更有动力完成
- 占便宜 = 超值感 = 购买动力

**实践方法**：
1. **提前给予**：给用户一些「已有」的东西，让他们不想浪费
2. **制造进度**：让用户感觉已经开始了，而不是从零开始
3. **占便宜感**：送小礼品，让用户觉得超值
4. **沉没成本**：利用人不想浪费的心理

**内容创作应用**：
- 📱 短视频选题：「为什么送2个印花比从零开始效果好？」

# 销售技巧

📅 [[2022-05-01]] 22:20:20

销售遇到一堆问问题，最后不成交的。怎么解决。谁提问谁掌握主动权，挖掘需求，解决问题是主要的。所以再别人问了3个问题之后。价值不到，价格不报。（先说价值，不然直接说价格别人心理价值锚点都是不到的）。讲清楚价值，搞清楚他的焦虑，他的问题梁靠谱#销售#客服#需求

---

### 🧠 深度解析

**核心洞察**：
这是一个关于「销售技巧」的实战经验。核心观点：**价值不到，价格不报**。先说价值，再说价格，否则价格锚点太低。

**底层逻辑**：
- 谁提问 = 谁掌握主动权
- 问问题的人 = 可能在白嫖
- 价值不到 = 价格报了也没用
- 先说价值 = 建立价格锚点 = 再报价格

**实践方法**：
1. **控制提问**：别人问3个问题后，开始反问
2. **挖掘需求**：搞清楚他的焦虑和问题
3. **先说价值**：建立价值认知，再报价格
4. **筛选客户**：问一堆问题不成交的，及时止损

**内容创作应用**：

**核心问题**：如何通过趣味文字印花设计，提高商品的销量和吸引力

**底层逻辑**：
- 趣味文字能够增加商品的情感共鸣，吸引年轻消费者的注意力

## 原始内容
关于创业项目的文章
标题：完美的创业项目，是怎么被想出来的？
作者：刘小排（昨天 19:22）

内容：
斯蒂芬·茨威格在《人类的群星闪耀时》中写了这么一句话：一个人生命中的最大幸运，莫过于在他的人生中途，即在他年富力强时发现了自己的人生使命。

符合"自己的人生使命"的创业项目，就是一个完美的项目。

一些圈友看到生财有术的「项目库」会产生一种心理上的排斥感：这些都不是我的使命啊，我都不想做。然后，啥也没做。

**核心洞察**：
这篇文章揭示了「完美项目幻觉」的创业误区。很多人陷入「先找完美项目再开干」的思维陷阱，结果啥也没做。**完美项目不是想出来的，是做出来的**。

- 完美项目 = 人生使命 = 高门槛
- 想找完美项目 = 拖延症 = 逃避行动
- 使命 = 做出来的 = 不是想出来的
- 先开干 = 试错 = 找到适合自己的

**创业真相**：
1. 没有完美项目，只有适合自己的项目
2. 先开干比想完美更重要

# 面试套项目：如何找到靠谱项目

**原始时间**：2022-06-07 11:21:53

**原文内容**：
什么项目才是好项目？分享人：比高。面试找工作绝对是个很好的路子，当然不是真去上班，而是通过这种方式去找到一些项目思路。

**核心方法**：
1. **简历**：完善简历吸引对方，但不能过于高调
2. **缺点**：一定要有缺点，比如"格局不够，聪明有余但毫无智慧"
3. **目的**：面试不是为了找工作，而是套关键信息，必须和老板或项目总负责人谈
4. **获取信息**：
   - 跟我配合的有多少个同事？（员工规模）
   - 还在持续招人吗？（增长空间）
   - 有没有提成？（项目成熟度）
   - 薪资结构怎么定的？（团队性质）
   - 提成怎么算的？（核心变现模式）
   - 流量从哪来？（流量口）

**核心洞察**：
啥叫靠谱项目？不是听别人吹牛逼，而是看这个模式放在市场验证后，是不是养活了这个公司，能养活且还能赚钱，就是靠谱的。

---

## 🧠 深度解析

**底层逻辑**：
- 面试 = 套项目信息的入口
- 和老板谈 = 获取关键信息
- 员工规模 + 招人计划 = 项目增长空间
- 提成明确 = 项目成熟

**实践方法**：
1. **完善简历**：吸引对方但不过于高调
2. **设定缺点**：让对方觉得需要"大哥"带
3. **锁定关键人**：只和老板或项目负责人谈
4. **提问清单**：员工规模、招人计划、提成、薪资结构、流量来源

**内容创作应用**：
- 📱 短视频选题：「面试套项目：一个被忽视的信息获取方式」
- 📝 长文选题：「如何通过面试判断项目靠不靠谱」
- 💬 金句卡片：「靠谱项目 = 市场验证 + 养活公司」
