**核心理念**：禁用词库 + 精准受众 + 非线性表达 = 真人味

**禁用词库**：
```
总之、不仅如此、关键在于、深入探讨、开启新篇章、更重要的是、值得注意的是、是一个强有力的工具、解锁潜力、极具吸引力的、令人惊叹的、全方位的、多维度的。
```

**受众画像四要素**：
1. 具体的人（谁）
2. 具体的烦恼（痛点）

**触发词**：
- `去AI味：[主题]` → 去机器味生成
