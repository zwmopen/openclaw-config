### 笔记系统架构（OrbitOS + claudesidian）

用户基于 GitHub MarsWang42/OrbitOS 和 heyits-noah/claudesidian 融合构建了笔记系统。

**核心经验**：
1. **不需要过度担心AI记不住**
   - 限制文件目录深度为3层
   - AI忘记时把路径重新写入记忆文件

2. **收件箱机制**
   - zip、图像、链接、pdf先放收件箱
   - 让AI出整理计划

3. **信息收录三大类**
   - 插件：Obsidian Web Clipper、HoverNotes
   - 微信：微信笔记同步助手（支持小红书视频转图文）
   - OpenClaw：专门啃难解析链接和视频

4. **Deep Research功能强大**
   - Gemini、Grok、OpenAI的Deep Research都可以用

5. **OpenClaw可以把信息录入和整理合并为一步**

6. **软链接机制**
   - `D:\Program Files\Obsidian\zwm\.zwm\OpenClaw配置\` → `D:\AI编程\openclaw\`
   - 在Obsidian里编写SOUL.md，OpenClaw立刻生效
