# 心跳扫描脚本（Python版）

每次心跳执行以下命令：

```powershell
python "D:\AI编程\openclaw\scripts\scan_v2.py" 100
```

**参数**：
- 100：每次扫描100个文件

**执行逻辑**：
1. 读取状态文件
2. 如果 status = "running"，执行扫描
3. 扫描100个文件，提取金句和概念
4. 更新状态文件
5. 每500个文件汇报一次

**状态文件**：`D:\Program Files\Obsidian\zwm\.zwm\02-ZWM 2.0 内容生产系统\01-内容生产\03-内容素材库\扫描状态.json`

**素材库**：
- 金句库：`D:\Program Files\Obsidian\zwm\.zwm\02-ZWM 2.0 内容生产系统\01-内容生产\03-内容素材库\金句库.md`
- 核心概念库：`D:\Program Files\Obsidian\zwm\.zwm\02-ZWM 2.0 内容生产系统\01-内容生产\03-内容素材库\核心概念库.md`

**金句识别规则**（7种模式）：
1. 对比型：不是A，而是B
2. 定义型：A的本质是B
3. 因果型：A决定B
4. 比喻型：A像B、A就是B
5. 行动型：不要A，要B
6. 洞察型：你赚不到认知以外的钱
7. 排比型：提升A，提升B，找到C

**长度要求**：5-100字
