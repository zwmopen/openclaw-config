# 工作本质是自我营销

📅 [[2022-04-14]] 08:49:24

工作需要主动，提供价值，交流本就是一种自我营销与展示

---

## 🧠 深度解析

**核心洞察**：
这是一个关于「职场本质」的深刻认知。工作不只是做事，更是「自我营销」。你说的「交流本就是一种自我营销与展示」，点破了职场的底层逻辑：**你的价值，取决于别人对你的认知**。

**底层逻辑**：
- 主动 = 被看见 = 有机会
- 被动 = 被忽略 = 没机会
- 交流 = 展示价值 = 建立认知
- 不交流 = 价值归零 = 没人知道你

**实践方法**：
1. **主动汇报**：每周主动向领导汇报一次工作进展
2. **价值外化**：把做的事说出去，不要默默无闻
3. **交流策略**：每次交流前想清楚「我要展示什么价值」
4. **建立标签**：让别人提到某个领域就想到你

**内容创作应用**：
- 📱 短视频选题：「为什么默默做事的人，永远升不了职？」
- 📝 长文选题：「职场真相：你的价值，取决于别人对你的认知」

# 钓鱼哲学与选择大于努力

📅 [[2022-04-16]] 16:12:54

钓鱼的第一原则是在有鱼的地方钓鱼活着的第一条原则就是不去会让自己死掉的地方。去合适的位置，找合适的时间，做合适的选择

---

## 🧠 深度解析

**核心洞察**：
这是一个关于「选择大于努力」的经典隐喻。钓鱼的第一原则不是「钓技多好」，而是「在有鱼的地方钓鱼」。人生的道理一样：**选择正确的位置，比努力更重要**。

**底层逻辑**：
- 有鱼的地方 = 有机会的地方
- 没鱼的地方 = 再努力也没用
- 不去会死的地方 = 风险控制
- 合适的位置 + 合适的时间 = 成功的一半

**三个核心原则**：
1. **位置原则**：在有鱼的地方钓鱼（选择大于努力）
2. **生存原则**：不去会让自己死掉的地方（风险控制）
3. **时机原则**：合适的位置 + 合适的时间（时机很重要）

**实践方法**：
1. **评估你的「鱼塘」**：你现在的地方，有鱼吗？
2. **换鱼塘**：如果没鱼，换一个地方
3. **风险清单**：列出会让你「死掉」的事，坚决不做
4. **时机判断**：现在是不是合适的时间？

**内容创作应用**：
- 📱 短视频选题：「为什么努力的人不一定成功？」
- 📝 长文选题：「钓鱼哲学：选择大于努力的三个原则」

**爆点话题**：
- 🔥 「你的鱼塘，有鱼吗？」
- 🔥 「为什么有些人再努力也没用？」

**底层逻辑**：
- 投资 = 现在资源 → 未来资源
- 活在未来 = 未来眼光 → 现在行为
- 合理配置 = 让资源产生最大价值
- 确定性 = 通过行动让未来更确定

**实践方法**：
1. **识别资源**：你有什么资源？（时间、金钱、精力、人脉）
2. **配置资源**：把资源投入到未来有价值的地方
3. **用未来眼光**：问「5年后，这个资源会变成什么？」
4. **增加确定性**：通过行动，让未来更确定

- 📱 短视频选题：「什么是真正的投资？」
