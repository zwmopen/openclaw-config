**每次心跳执行**：

### 1. 检查状态文件
```powershell
$s = Get-Content "D:\Program Files\Obsidian\zwm\.zwm\01-个人成长内容生产\04-素材库\心跳扫描状态.json" | ConvertFrom-Json
```

### 2. 判断任务状态
- **status = "running"** → 执行扫描（每次100个文件）
- **status = "idle"** → 返回 HEARTBEAT_OK
- **status = "pending"** → 检查新增文件，决定是否启动扫描

### 3. 执行扫描（当status = running时）
```powershell
$dirs = @("01-flomo闪念", "02-我的幕布", "03-微信读书", "04-和AI聊天的库", "05-印象笔记", "06-飞书批量导出")
$kbPath = "D:\Program Files\Obsidian\zwm\.zwm\个人知识库"
$s = Get-Content "D:\Program Files\Obsidian\zwm\.zwm\01-个人成长内容生产\04-素材库\心跳扫描状态.json" | ConvertFrom-Json
if ($s.status -eq "running") {
    $currentDir = $s.currentDir
    $files = Get-ChildItem "$kbPath\$currentDir" -Recurse -File -Filter *.md | Select-Object -Skip $s.scanned -First 100
    if ($files.Count -eq 0) {
        $idx = [array]::IndexOf($dirs, $currentDir)
        if ($idx -lt $dirs.Count - 1) {
            $s.currentDir = $dirs[$idx + 1]
            $s.scanned = 0
        } else {
            $s.status = "idle"
        }
    } else {
        $q = 0; foreach ($f in $files) { $c = Get-Content $f.FullName -Encoding UTF8 -Raw; if ($c -match '(关键|核心|重点|要点|本质|规律|原则|方法|技巧|秘诀)') { $q++ } }
        $s.scanned += $files.Count
        $s.quotes.completed += $q
        $s.quotes.progress = [math]::Round($s.quotes.completed / $s.quotes.target * 100, 1).ToString() + '%'
    }
    $s.lastScan = Get-Date -Format 'yyyy-MM-dd HH:mm'
    $s | ConvertTo-Json -Depth 10 | Set-Content "D:\Program Files\Obsidian\zwm\.zwm\01-个人成长内容生产\04-素材库\心跳扫描状态.json" -Encoding UTF8
}
```

### 4. 汇报规则
- 每扫描500个文件 → 发送进度汇报
- 扫描完成 → 发送完成通知
- 无任务 → 静默返回 HEARTBEAT_OK

### 5. 触发扫描
用户说以下命令启动扫描：
- `开始扫描` → 设置 status = "running"
- `继续扫描` → 继续上次进度
