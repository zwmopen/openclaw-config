# 金句识别规则（2026-03-08 更新）

## 什么是金句？

**金句**是具有传播力、感染力、记忆点的句子。它不是普通的引用，而是：
- **有哲理**：揭示人生真相、商业规律、人性本质
- **有洞察**：打破认知、引发思考、产生共鸣
- **易传播**：短小精悍、朗朗上口、值得分享

---

## 金句的核心特征

### 1. 长度特征
- **最佳长度**：10-50字
- **可接受**：5-100字
- **排除**：超过100字的段落

### 2. 内容特征
- ✅ **揭示本质**：A的本质是B、A决定B
- ✅ **打破认知**：不是A，而是B
- ✅ **引发共鸣**：说出大家想说的话
- ✅ **指导行动**：做A，不做B
- ✅ **洞察规律**：A的边界是B的边界

### 3. 修辞特征
- **对比**：不是A，而是B / 不是A，是B
- **对仗**：有A，有B / A决定B，B影响A
- **比喻**：A像B / A就是B / 把A比作B
- **排比**：提升A，提升B，找到C
- **反转**：表面是A，实际是B
- **因果**：因为A，所以B / A决定B

## 金句识别模式（正则表达式）

### 模式1：对比型（最常见）
```regex
不是[，。、；：].*而是[，。、；：]
不是[，。、；：].*是[，。、；：]
不[，。、；：].*而[，。、；：]
```

### 模式2：定义型
```regex
.+的本质是.+
.+决定.+
.+是.+(最值钱|最重要|最关键|核心|关键)
```

### 模式3：因果型
```regex
.+决定.+
.+是.+(变现|结果|前提|基础)
.+的边界是.+的边界
```

### 模式4：比喻型
```regex
.+像.+
.+就是.+
把.+比作.+
.+如同.+
```

### 模式5：行动型
```regex
(不要|别|避免).+，(要|做|应该).+
(做|做对|选择).+，(不做|避免|放弃).+
```

### 模式6：洞察型
```regex
.+是.+(唯一|最好|最有效|最简单).+(方法|路径|方式)
你(赚不到|得不到|无法).+认知以外的.+
```

### 模式7：排比型
```regex
(提升|增加|减少|做|有).+，(提升|增加|减少|做|有).+，(提升|增加|减少|做|有).+
```

## 金句识别完整规则（PowerShell）

```powershell
# 金句识别函数
function Is-GoldenQuote {
    param([string]$sentence)
    
    # 长度检查
    if ($sentence.Length -lt 5 -or $sentence.Length -gt 100) {
        return $false
    }
    
    # 排除简单陈述（没有修辞）
    if ($sentence -notmatch '(不是|而是|决定|本质|核心|关键|像|就是|做|要|别|避免|提升|增加|减少|找到|选择|大于|小于|最重要|最值钱|唯一|最好)') {
        return $false
    }
    
    # 模式匹配
    $patterns = @(
        # 对比型
        '不是.+而是',
        '不是.+是',
        '不.+而.+',
        
        # 定义型
        '.+的本质是',
        '.+决定.+',
        '.+是.+最.+',
        '.+是.+核心',
        '.+是.+关键',
        
        # 因果型
        '.+的边界是.+的边界',
        '.+是.+变现',
        '.+是.+结果',
        
        # 比喻型
        '.+像.+',
        '.+就是.+',
        '把.+比作',
        
        # 行动型
        '(不要|别).+要',
        '做.+不做',
        '选择.+放弃',
        
        # 洞察型
        '你(赚不到|得不到|无法).+认知',
        '.+是.+唯一.+方法',
        
        # 排比型
        '(提升|做|有).+，(提升|做|有).+，(提升|做|有)'
    )
    
    foreach ($pattern in $patterns) {
        if ($sentence -match $pattern) {
            return $true
        }
    }
    
    return $false
}
```

## 金句 vs 非金句示例

### ✅ 金句（符合规则）
- "不是你付出了多少，是别人感知到了多少。" - **对比型**
- "营销的本质：不是你有什么，是别人感知到什么。" - **定义+对比型**
- "你要去有鱼的地方钓鱼。" - **比喻型**
- "认知决定财富，财富是认知的变现。" - **因果型**
- "不要all in，不要加自己承受不起的杠杆。" - **行动型**
- "你赚不到认知以外的钱。" - **洞察型**

### ❌ 非金句（不符合规则）
- "今天天气很好。" - 简单陈述，无哲理
- "感知价值可以被塑造，它可以完全和实际价值错位。" - 太长，没有修辞
- "2026年3月4日采集。" - 纯描述
- "标题：如何做小红书" - 标题，不是金句

## 扫描流程

1. **提取句子**：从笔记中提取所有句子（按句号、问号、感叹号分割）
2. **长度过滤**：保留5-100字的句子
3. **模式匹配**：使用正则表达式匹配金句模式
4. **人工审核**：最终由人工确认是否真的是金句
