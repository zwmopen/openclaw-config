## 操作步骤

### 阶段1：飞书实名认证（今天完成）
1. 打开飞书App
2. 点击头像 → 设置与隐私 → 账号与安全
3. 找到实名认证 → 选择「个人实名认证」
4. 按提示完成：身份证拍照 + 人脸核验
5. 等待审核（通常几分钟内通过）

### 阶段2：提交申请（明天完成）

**OpenClaw助手（本地）**：
1. 登录 https://open.feishu.cn/
2. 点击「应用」→ 找到「OpenClaw助手」→ 点击进入
3. 左侧菜单找到「API 配额」
4. 点击「申请提升配额」
5. 选择「无限调用额度（限时免费，3.31截止）」
6. 粘贴上面的申请材料
7. 提交申请

**腾讯小龙虾OpenClaw（腾讯云）**：
重复上述步骤，选择「腾讯小龙虾open claw」应用

### 阶段3：验证额度（申请通过后）
1. 回到飞书开放平台 → 进入应用 → 「API 配额」
