# 飞书机器人配置指南

## 一、创建飞书应用

### 1.1 访问飞书开放平台
1. 打开浏览器，访问：https://open.feishu.cn/
2. 使用您的飞书账号登录
3. 点击右上角"开发者控制台"

### 1.2 创建企业自建应用
1. 点击"创建企业自建应用"
2. 填写应用信息：
   - **应用名称**：例如"OpenClaw助手"
   - **应用描述**：例如"AI智能助手机器人"
   - **应用图标**：上传一个图标（可选）
3. 点击"创建"

## 二、配置应用权限（快速方法）

### 2.1 使用JSON批量导入权限（推荐）

**步骤：**

1. 在飞书开放平台，进入您的应用
2. 点击左侧菜单"权限管理"
3. 找到 **"批量导入"** 或 **"导入权限配置"** 按钮
4. 打开文件 [feishu_permissions.json](./feishu_permissions.json)
5. 复制全部内容
6. 粘贴到导入框中
7. 点击 **"确认导入"**
8. 检查权限列表，确认所有权限已添加

### 2.2 权限类型说明

**重要：** 飞书有两种权限类型，请根据需求选择：

#### 应用身份权限（tenant_access_token）
- **用途**：应用以自己的身份访问数据
- **特点**：不需要用户授权，适合机器人场景
- **推荐**：机器人应用主要使用此类型

#### 用户身份权限（user_access_token）
- **用途**：应用代表用户访问数据
- **特点**：需要用户授权，可访问用户私有数据
- **推荐**：需要访问用户私有数据时使用

### 2.3 JSON配置详解

```json
{
  "scopes": {
    "tenant": [
      "im:message",                    // 获取与发送消息
      "im:message:send_as_bot",        // 以应用身份发消息
      "im:message:receive_as_bot",     // 接收@机器人消息
      "im:chat",                       // 获取群组信息
      "im:chat:readonly",              // 获取群组信息（只读）
      "im:chat.member:readonly",       // 获取群成员列表
      "contact:user.base:readonly",    // 获取用户基本信息
      "contact:user.employee_id:readonly", // 获取用户员工ID
      "im:resource"                    // 获取与上传图片或文件
    ],
    "user": [
      "im:message",                    // 获取与发送消息
      "im:chat",                       // 获取群组信息
      "im:chat:readonly",              // 获取群组信息（只读）
      "contact:user.base:readonly"     // 获取用户基本信息
    ]
  }
}
```

**说明：**
- **tenant**：应用身份权限（机器人主要使用）
- **user**：用户身份权限（可选，根据需求添加）

### 2.4 手动添加权限（备选方法）

如果无法使用JSON导入，可以手动添加权限：

#### 应用身份权限（tenant_access_token）

**消息相关：**
1. `im:message` - 获取与发送单聊、群组消息
2. `im:message:send_as_bot` - 以应用身份发消息
3. `im:message:receive_as_bot` - 接收群聊中@机器人消息
4. `im:chat` - 获取群组信息
5. `im:chat:readonly` - 获取群组信息（只读）
6. `im:chat.member:readonly` - 获取群成员列表

**用户相关：**
7. `contact:user.base:readonly` - 获取用户基本信息
8. `contact:user.employee_id:readonly` - 获取用户员工ID

**资源相关：**
9. `im:resource` - 获取与上传图片或文件资源

#### 用户身份权限（user_access_token）- 可选

1. `im:message` - 获取与发送消息
2. `im:chat` - 获取群组信息
3. `im:chat:readonly` - 获取群组信息（只读）
4. `contact:user.base:readonly` - 获取用户基本信息

**注意：** 部分权限需要管理员审批

## 三、配置事件订阅

### 3.1 启用事件订阅
1. 在应用管理页面，找到"事件订阅"
2. 开启"启用事件订阅"

### 3.2 配置请求网址
- **请求网址**：`https://您的服务器地址/webhook/feishu`
- 注意：需要公网可访问的HTTPS地址

### 3.3 订阅事件类型
添加以下事件：
- `im.message.receive_v1` - 接收消息
- `im.chat.member.added_v1` - 群成员增加
- `im.chat.member.deleted_v1` - 群成员减少

## 四、获取应用凭证

### 4.1 获取App ID和App Secret
1. 在应用管理页面，找到"凭证与基础信息"
2. 复制以下信息：
   - **App ID**：应用的唯一标识
   - **App Secret**：应用的密钥（点击"查看"获取）

### 4.2 获取Verification Token
1. 在"事件订阅"页面
2. 找到"Verification Token"
3. 复制该Token

## 五、发布应用

### 5.1 创建版本
1. 在应用管理页面，点击"版本管理与发布"
2. 点击"创建版本"
3. 填写版本号和更新说明
4. 点击"保存"

### 5.2 申请发布
1. 点击"申请发布"
2. 填写申请说明
3. 等待管理员审批

### 5.3 启用应用
审批通过后：
1. 在飞书管理后台找到"工作台"
2. 找到您的应用
3. 点击"启用"

## 六、配置机器人功能

### 6.1 启用机器人能力
1. 在应用管理页面，找到"应用功能"
2. 点击"机器人"
3. 开启"启用机器人"

### 6.2 配置机器人信息
- **机器人名称**：显示在群聊中的名称
- **机器人描述**：功能描述
- **机器人头像**：上传头像图片

### 6.3 配置消息卡片（可选）
1. 在"消息卡片"页面
2. 创建消息卡片模板
3. 用于富文本消息展示

## 七、添加机器人到群聊

### 7.1 在群聊中添加
1. 打开飞书群聊
2. 点击群设置（右上角齿轮图标）
3. 选择"群机器人"
4. 点击"添加机器人"
5. 选择您创建的机器人

### 7.2 使用机器人
- 在群聊中@机器人名称，发送消息
- 机器人会自动回复

## 八、安全配置

### 8.1 IP白名单（可选）
如果需要限制访问来源：
1. 在"安全设置"页面
2. 添加服务器IP到白名单

### 8.2 数据加密
- 所有通信使用HTTPS加密
- 消息内容使用Verification Token验证

飞书开放平台提供：

## 十、常见问题

### Q1: 权限申请一直待审批？
**A**: 联系飞书管理员，在管理后台审批权限申请

### Q2: 机器人不回复消息？
**A**: 检查以下几点：
- 事件订阅是否正确配置
- 服务器是否能正常接收请求
- 权限是否已开通

### Q3: 如何获取用户Open ID？
**A**: 通过消息事件获取，或调用用户信息API

### Q4: 如何发送富文本消息？
**A**: 使用消息卡片功能，创建富文本模板

### Q5: tenant和user权限有什么区别？
**A**: 
- **tenant权限**：应用以自己的身份访问数据，适合机器人场景
- **user权限**：应用代表用户访问数据，需要用户授权

### Q6: 我应该选择哪种权限类型？
**A**: 
- 如果是机器人应用，主要使用 **tenant权限**
- 如果需要访问用户私有数据，添加 **user权限**
- 推荐两种都添加，确保功能完整
