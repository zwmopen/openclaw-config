# 飞书API使用次数统计查看方法

---

## 📊 查看步骤

### 方法一：飞书开放平台

1. 打开飞书开放平台：https://open.feishu.cn/
2. 登录你的飞书账号
3. 点击右上角 **控制台**
4. 选择你的应用
5. 点击左侧菜单 **数据统计**
6. 查看 **API调用详情**

### 方法二：飞书开发者后台

1. 打开飞书开发者后台：https://open.feishu.cn/app
2. 选择你的应用
3. 点击 **应用详情**
4. 找到 **数据统计** 或 **API调用统计**
5. 查看调用次数、成功率等数据

### 方法三：应用管理页面

1. 打开飞书客户端
2. 点击左上角头像 → **管理后台**
3. 找到你的应用
4. 查看 **使用统计** 或 **数据报表**

---

## 📈 统计内容

通常可以看到：

| 数据 | 说明 |
|------|------|
| **API调用次数** | 每月调用总数 |
| **调用成功率** | 成功/失败比例 |
| **调用趋势** | 每日/每周/每月趋势图 |
| **接口详情** | 各接口调用次数 |
| **错误日志** | 失败调用原因 |

## ⚠️ 注意事项

1. **数据延迟**：统计数据可能有几分钟到几小时的延迟
2. **统计周期**：通常按自然月统计，每月1号重置
3. **额度提醒**：可以在控制台设置额度提醒，避免超额
