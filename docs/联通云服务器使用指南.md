## 三、配置OpenClaw

### 1. 设置模型提供商

```bash
# 添加硅基流动
openclaw config set models.providers.siliconflow.baseUrl "https://api.siliconflow.cn/v1"
openclaw config set models.providers.siliconflow.apiKey "sk-hyaakxkupozeqeisekcvnvterzifvgcjconbghtmxgrjufke"

# 添加联通元景
openclaw config set models.providers.unicom.baseUrl "https://maas-api.ai-yuanjing.com/openapi/compatible-mode/v1"
openclaw config set models.providers.unicom.apiKey "sk-33b2451706fb4098850b14a9dfbb5827"

# 添加智谱AI官网
openclaw config set models.providers.zhipu.baseUrl "https://open.bigmodel.cn/api/paas/v4/"
openclaw config set models.providers.zhipu.apiKey "aaba2e5167954bac8fb2188d882c40db.qP2yJKa6qjLA16XA"
```

### 2. 设置默认模型

```bash
openclaw config set agents.defaults.model.primary "siliconflow/deepseek-ai/DeepSeek-V3"
```

### 3. 开放网关端口

```bash
# 开放18789端口
sudo ufw allow 18789
# 或
sudo iptables -A INPUT -p tcp --dport 18789 -j ACCEPT
```

### 4. 启动网关

```bash
# 前台运行
openclaw gateway

# 后台运行
nohup openclaw gateway > openclaw.log 2>&1 &
```
