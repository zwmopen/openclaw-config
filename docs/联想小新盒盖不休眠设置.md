# 联想小新盒盖不休眠设置指南

> 适用：Win10/Win11 通用
> 目标：盒盖后依然正常运行、不睡眠、不断网、程序继续跑

---

## 一、核心设置：合盖不睡眠（必做）

1. 按 `Win+R` → 输入 `powercfg.cpl` → 回车，打开电源选项
2. 左侧点 **选择关闭盖子的功能**
3. 找到 **关闭盖子时**：
   - 接通电源：选 **不采取任何操作**
   - 使用电池：选 **不采取任何操作**（想省电可设"睡眠"，但盒盖会停）
4. 点 **保存修改**

---

## 二、配套设置：锁屏/盒盖也不睡眠（防断）

1. 回到电源选项 → 点当前计划（如"平衡"）→ **更改计划设置**
2. 接通电源时：
   - 关闭显示器：**从不**
   - 使计算机进入睡眠状态：**从不**
3. 点 **更改高级电源设置**
4. 展开 **睡眠**：
   - 允许混合睡眠：**关闭**
   - 允许使用唤醒定时器：**启用**
   - 在此时间后睡眠：**从不**
5. 展开 **硬盘** → 关闭硬盘：**从不**
6. 展开 **无线适配器设置** → 节能模式：**最高性能**（防WiFi断）
7. 展开 **电源按钮和盖子** → 盖子关闭操作：确认是 **不采取任何操作**
8. 确定保存

---

## 三、关闭现代待机（小新新机型必做）

小新很多新款默认开现代待机（S0低功耗），盒盖会断网/停任务：

1. 按 `Win+X` → 终端（管理员）/命令提示符（管理员）
2. 输入：
   ```powershell
   powercfg /a
   ```
3. 若显示 `Standby (S0 Low Power Idle) Network Connected`，说明开了现代待机
4. 关闭方法（Win11专业版/企业版）：
   - `Win+R` → `gpedit.msc` → 计算机配置→管理模板→系统→电源管理→电源按钮和盖子
   - 双击 **启动时启用现代待机** → 选 **已禁用** → 确定
   - 重启电脑生效

---

## 四、应用后台保活（让软件不被挂起）

1. `Win+I` → 隐私和安全 → 后台应用
2. 总开关：**开启**
3. 找到要保活的软件（下载、远程、脚本等）：设为 **始终允许后台运行**

---

## 五、测试与使用

- 直接合盖，电脑会继续运行、下载/渲染/远程都不停
- 按 `Win+L` 锁屏，也不影响运行
- 不要点「睡眠/休眠」，只合盖/锁屏即可

---

## 常见问题

| 问题 | 解决方案 |
|------|---------|
| 还是断网 | 检查杀毒/安全软件，把程序加入白名单 |
| 盒盖黑屏但风扇转 | 正常，屏幕关了但系统在跑 |
| 电池掉电快 | 插电用；电池模式建议设"睡眠"更省电 |

---

## 快速命令（管理员权限）

```powershell
# 检查当前电源设置
powercfg /a

# 设置盒盖不睡眠（接通电源）
powercfg /setacvalueindex SCHEME_CURRENT 4f971e89-eebd-4455-a8de-9e59040e7347 5ca83367-6e45-459f-a27b-476b1d01c936 0

# 设置盒盖不睡眠（电池）
powercfg /setdcvalueindex SCHEME_CURRENT 4f971e89-eebd-4455-a8de-9e59040e7347 5ca83367-6e45-459f-a27b-476b1d01c936 0

# 应用设置
powercfg /setactive SCHEME_CURRENT
```
