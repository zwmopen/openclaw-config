# 天气技能

获取当前天气和天气预报。

## 使用场景

✅ **使用这个技能：**

- "今天天气怎么样？"
- "今天/明天会下雨吗？"
- "[城市]的温度"
- "一周天气预报"
- 旅行前查天气

❌ **不使用这个技能：**

- 历史天气数据 → 用天气档案/API
- 气候分析或趋势 → 用专业数据源
- 超本地微气候数据 → 用本地传感器
- 严重天气预警 → 查官方NWS源
- 航空/海洋天气 → 用专业服务（METAR等）

## 位置

查询天气时总是包含城市、地区或机场代码。

## 命令

### 当前天气

```bash
# 单行摘要
curl "wttr.in/London?format=3"

# 详细当前天气
curl "wttr.in/London?0"

# 指定城市
curl "wttr.in/New+York?format=3"

# 中文城市
curl "wttr.in/北京?format=3"
curl "wttr.in/杭州?format=3"
```

### 预报

```bash
# 3天预报
curl "wttr.in/London"

# 一周预报
curl "wttr.in/London?format=v2"

# 特定日期（0=今天，1=明天，2=后天）
curl "wttr.in/London?1"
```

### 格式选项

```bash
# 单行
curl "wttr.in/London?format=%l:+%c+%t+%w"

# JSON输出
curl "wttr.in/London?format=j1"

# PNG图片
curl "wttr.in/London.png"
```

### 格式代码

- `%c` — 天气状况表情
- `%t` — 温度
- `%f` — "体感"
- `%w` — 风
- `%h` — 湿度
- `%p` — 降水
- `%l` — 位置

## 快速响应

**"今天天气怎么样？"**

```bash
curl -s "wttr.in/London?format=%l:+%c+%t+(体感+%f),+%w+风,+%h+湿度"
```

**"会下雨吗？"**

```bash
curl -s "wttr.in/London?format=%l:+%c+%p"
```

**"周末预报"**

```bash
curl "wttr.in/London?format=v2"
```

## 注意事项

- 不需要API密钥（使用wttr.in）
- 有请求限制，不要频繁请求
- 支持大多数全球城市
- 支持机场代码：`curl wttr.in/ORD`
