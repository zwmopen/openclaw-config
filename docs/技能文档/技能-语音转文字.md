# 语音转文字（Whisper）

使用 `whisper` 在本地转录音频。

## 快速开始

```bash
# 转录音频文件
whisper /path/audio.mp3 --model medium --output_format txt --output_dir .

# 翻译音频（转成英文）
whisper /path/audio.m4a --task translate --output_format srt

# 中文音频转文字
whisper /path/audio.mp3 --language zh --model medium --output_format txt
```

## 模型选择

| 模型 | 大小 | 速度 | 精度 |
|------|------|------|------|
| tiny | 39MB | 最快 | 最低 |
| base | 74MB | 快 | 低 |
| small | 244MB | 中等 | 中等 |
| medium | 769MB | 慢 | 高 |
| large | 1550MB | 最慢 | 最高 |
| turbo | 809MB | 快 | 高（默认） |

## 参数说明

| 参数 | 说明 |
|------|------|
| `--model` | 模型大小，默认turbo |
| `--language` | 语言代码（zh=中文，en=英文） |
| `--output_format` | 输出格式：txt, srt, vtt, json |
| `--output_dir` | 输出目录 |
| `--task` | 任务类型：transcribe（转录），translate（翻译成英文） |

## 使用场景

✅ **使用这个技能：**

- "把这个音频转成文字"
- "语音识别"
- "转录录音"
- "视频字幕提取"

❌ **不使用这个技能：**

- 实时语音识别 → 用其他服务
- 在线语音转文字 → 用API版本

## 注意事项

- 首次运行会自动下载模型到 `~/.cache/whisper`
- 模型越大越精确，但越慢
- 中文音频建议指定 `--language zh`
- 不需要API密钥，完全本地运行

## 安装

```bash
# macOS
brew install openai-whisper

# Windows (需要先安装Python)
pip install openai-whisper

# 验证安装
whisper --help
```
