# 飞书权限工具

单一工具 `feishu_perm` 用于管理文件/文档权限。

## 操作

### 列出协作者

```json
{ "action": "list", "token": "ABC123", "type": "docx" }
```

返回：带member_type、member_id、perm、name的成员。

### 添加协作者

```json
{
  "action": "add",
  "token": "ABC123",
  "type": "docx",
  "member_type": "email",
  "member_id": "user@example.com",
  "perm": "edit"
}
```

### 移除协作者

```json
{
  "action": "remove",
  "token": "ABC123",
  "type": "docx",
  "member_type": "email",
  "member_id": "user@example.com"
}
```

## Token类型

| 类型 | 描述 |
|------|------|
| `doc` | 旧版文档 |
| `docx` | 新版文档 |
| `sheet` | 电子表格 |
| `bitable` | 多维表格 |
| `folder` | 文件夹 |
| `file` | 上传的文件 |
| `wiki` | Wiki节点 |
| `mindnote` | 思维导图 |

## 成员类型

| 类型 | 描述 |
|------|------|
| `email` | 邮箱地址 |
| `openid` | 用户open_id |
| `userid` | 用户user_id |
| `unionid` | 用户union_id |
| `openchat` | 群聊open_id |
| `opendepartmentid` | 部门open_id |

## 权限级别

| 权限 | 描述 |
|------|------|
| `view` | 仅查看 |
| `edit` | 可编辑 |
| `full_access` | 完全访问（可管理权限） |

## 示例

通过邮箱分享文档：

```json
{
  "action": "add",
  "token": "doxcnXXX",
  "type": "docx",
  "member_type": "email",
  "member_id": "alice@company.com",
  "perm": "edit"
}
```

与群组分享文件夹：

```json
{
  "action": "add",
  "token": "fldcnXXX",
  "type": "folder",
  "member_type": "openchat",
  "member_id": "oc_xxx",
  "perm": "view"
}
```

## 配置

```yaml
channels:
  feishu:
    tools:
      perm: true # 默认：false（禁用）
```

**注意：** 此工具默认禁用，因为权限管理是敏感操作。如需要请明确启用。

## 权限

需要：`drive:permission`
