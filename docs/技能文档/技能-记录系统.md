# 记录系统技能

> 快速记录选题、想法、笔记，并自动分类处理

## 触发词

| 关键词 | 动作 | 存放位置 |
|--------|------|----------|
| `记录选题` | 保存选题，标注日期 | `选题管理/今日选题/` |
| `记录想法` | 保存原始想法 | `内容素材库/flomo闪念笔记/` |
| `记录并深化` | 保存 + 扩展分析 | `内容素材库/深化笔记/` |
| `记录并变选题` | 保存 + 提炼成选题 | `选题管理/待评估选题/` |
| `记录并梳理` | 保存 + 结构化整理 | `内容素材库/梳理笔记/` |
| `记录素材` | 保存素材 | `内容素材库/素材/` |
| `记录灵感` | 快速记录一句话 | `内容素材库/flomo闪念笔记/` |

---

## 详细说明

### 1. 记录选题

**触发词：** `记录选题` / `添加选题` / `新选题`

**动作：**
- 在 `ZWM 2.0 内容生产系统/01-内容生产/选题管理/今日选题/` 创建文件
- 文件名：`选题标题.md`
- 自动添加日期标签

**格式：**
```markdown
# {选题标题}

- 分类：{自动判断或用户指定}
- 来源：{用户提供的来源}
- 记录时间：{当前日期}
- 优先级：⭐⭐⭐

## 核心角度

{用户提供的选题描述}

## 可执行方向

- [ ] 方向1
- [ ] 方向2

## 备注

{其他信息}
```

---

### 2. 记录想法

**触发词：** `记录想法` / `记个想法` / `想法`

**动作：**
- 在 `ZWM 2.0 内容生产系统/01-内容生产/内容素材库/flomo闪念笔记/` 创建文件
- 文件名：`{日期时间}_{想法关键词}.md`
- 保持原始表达，不做修改

**格式：**
```markdown
# {想法关键词}

{用户的原始想法}

---
记录时间：{当前日期时间}
标签：#想法 #闪念
```

---

### 3. 记录并深化

**触发词：** `记录并深化` / `深化这个想法` / `扩展一下`

**动作：**
- 先保存原始内容
- 然后进行深度扩展分析
- 存放在 `内容素材库/深化笔记/`

**格式：**
```markdown
# {标题}

## 原始想法

{用户的原始内容}

## 深化分析

### 为什么重要？
{分析}

### 可以怎么做？
{建议}

### 可能的结果
{预测}

## 延伸思考

{进一步扩展}

---
记录时间：{当前日期时间}
标签：#深化 #分析
```

---

### 4. 记录并变选题

**触发词：** `记录并变选题` / `变成选题` / `转选题`

**动作：**
- 分析内容，提炼选题角度
- 评估选题价值
- 存放在 `选题管理/待评估选题/`

**格式：**
```markdown
# 选题：{提炼的标题}

## 来源

{原始内容}

## 选题角度

1. **角度一：{角度名}**
   - 切入点：
   - 目标人群：
   - 预期效果：

2. **角度二：{角度名}**
   - 切入点：
   - 目标人群：
   - 预期效果：

## 选题评估

- 热度预测：⭐⭐⭐⭐
- 执行难度：⭐⭐
- 素材储备：{是否需要补充素材}

## 下一步

- [ ] 收集相关素材
- [ ] 确定最终角度
- [ ] 开始创作

---
记录时间：{当前日期时间}
来源标签：#选题转化
```

---

### 5. 记录并梳理

**触发词：** `记录并梳理` / `梳理一下` / `整理一下`

**动作：**
- 结构化整理内容
- 提取关键点
- 存放在 `内容素材库/梳理笔记/`

**格式：**
```markdown
# {主题}

## 一句话总结

{核心观点}

## 核心要点

1. {要点1}
2. {要点2}
3. {要点3}

## 关键细节

- {细节1}
- {细节2}

## 自媒体选题角度

1. {选题角度1}
2. {选题角度2}

---
记录时间：{当前日期时间}
标签：#梳理 #结构化
```

---

### 6. 记录素材

**触发词：** `记录素材` / `保存素材` / `存素材`

**动作：**
- 在 `内容素材库/素材/` 创建文件
- 分类保存

**格式：**
```markdown
# {素材标题}

## 来源

- 原文链接：{链接}
- 来源平台：{平台}
- 获取时间：{日期}

## 内容

{素材内容}

## 可用场景

- 场景1：
- 场景2：

---
标签：#素材 #{分类}
```

### 7. 记录灵感

**触发词：** `记录灵感` / `灵感` / `闪念`

**动作：**


```
用户：记录并深化：我发现很多人不会休息，休息也是需要学习的
```

## 注意事项

1. **文件名**：自动从内容提取关键词，避免特殊字符
2. **日期格式**：YYYY-MM-DD HH:mm
3. **标签规范**：统一用小写，用连字符连接
4. **不要重复创建**：先检查是否存在相同内容
5. **保持原意**：用户的内容不做过度修改
