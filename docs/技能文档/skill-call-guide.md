## 💡 精准表达示例

### ❌ 浪费token的说法
```
"那个，我想让你帮我整理一下这个笔记，就是按照你之前说的那个格式，然后把它整理好，放到那个文件夹里..."
```

```
"梳理笔记"
```

## 🔧 自动识别规则

Trae会自动识别你的意图，不需要精确匹配：

- 说"记录"相关的 → 笔记技能
- 说"任务/待办"相关的 → 任务技能
- 说"搜索/解析"相关的 → 网络技能
- 说"技能/插件"相关的 → 系统技能

**核心：直接说你想要什么，不用解释。**
