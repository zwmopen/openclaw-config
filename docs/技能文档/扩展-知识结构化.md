# 知识结构化技能

将任意Markdown笔记、幕布思维导图、碎片知识梳理成统一格式的结构化文档。

## 核心功能

1. **知识结构化**：将碎片化知识整理成系统化文档
2. **学习资料制作**：创建便于学习和复习的结构化内容
3. **自媒体选题**：将知识转化为可用于创作的内容素材

## 输出格式（10节结构）

每个结构化文档包含以下10个部分：

```
## 一、概念定义
核心概念解释，一句话说清楚是什么

## 二、核心要点
表格形式呈现关键信息，包含要点、说明、实践方法

## 三、实践模板
可直接使用的框架、清单、流程

## 四、知识关联
相关概念和来源，表格形式

## 五、应用场景
实际应用领域，列举4-6个场景

## 六、注意事项
需要避免的坑，关键提醒

## 七、核心金句
精华语录，原文中的精彩句子

## 八、体系图
ASCII可视化结构图

## 九、实践建议
短中长期行动指南

## 十、检查清单
自检问题，用于验证理解程度
```

## 使用方法

### 输入
- Markdown文件路径
- 或直接提供笔记内容

### 输出
- 在原文件同目录创建 `原文件名（已梳理）.md`
- 保持原文件不变

### 处理流程

```
1. 读取原始文件内容
2. 分析内容结构和核心概念
3. 提取关键要点和金句
4. 创建实践模板和检查清单
5. 绘制ASCII体系图
6. 生成结构化文档
7. 保存到原文件同目录
```

## 适用场景

| 场景 | 说明 |
|------|------|
| 个人知识管理 | 将碎片笔记整理成系统化知识库 |
| 学习资料制作 | 创建便于复习的学习文档 |
| 自媒体创作 | 将知识转化为可用的选题素材 |
| 读书笔记 | 将读书心得结构化输出 |
| 课程整理 | 将课程内容整理成笔记 |

## 示例

**输入文件**：`如何更好的决策.md`

**输出文件**：`如何更好的决策（已梳理）.md`

包含：
- 决策的定义和核心概念
- 决策工具表格（SWOT、决策矩阵等）
- 决策流程模板
- 相关思维模型
- 应用场景
- 注意事项
- 核心金句
- 决策体系图
- 短中长期实践建议
- 决策质量检查清单

## 批量处理

支持批量处理整个目录：

```
1. 扫描目录下所有.md文件
2. 过滤已处理的文件（含"已梳理"后缀）
3. 分批处理（每批4-6个文件）
4. 显示处理进度
5. 完成后统计处理数量
```

## 注意事项

1. **保持原意**：不改变原文核心观点
2. **补充完善**：可适当补充相关知识和实践方法
3. **统一格式**：严格遵循10节结构
4. **中文输出**：保持中文语言
5. **函数注释**：如涉及代码，添加函数级注释

## 调用方式

用户可以说：
- "帮我梳理这个文件"
- "把这个笔记结构化"
- "整理这个目录下的所有笔记"
- "创建学习资料"
- "生成自媒体选题素材"
