# 使用Trae的标准与技巧

## 背景

需要制定一套使用Trae的标准，包括提示词和技巧，让交互更高效。

---

## 记录模式

| 我说的 | Trae做的 |
|--------|---------|
| 记录选题 | 放到选题文件夹，标注日期 |
| 记录想法 | 放到想法文件夹，原样记录 |
| 记录并深化 | 记录 + 扩展思考 + 提炼认知 |
| 记录并变成选题 | 记录 + 分析选题角度 |
| 记录并梳理 | 记录 + 结构化整理 |
