# 技能-Obsidian写入

> 向 Obsidian 笔记库写入内容

---

## 功能

向 Obsidian 笔记库写入、追加、读取 Markdown 文件。

---

## 操作

| 操作 | 说明 |
|------|------|
| `write` | 写入文件（覆盖） |
| `append` | 追加内容到文件末尾 |
| `read` | 读取文件内容 |

---

## 配置

笔记库路径在技能中配置，或使用默认路径。

**当前配置的路径：**
```
D:\Program Files\Obsidian\zwm\.zwm\
```

---

## 示例

### 写入笔记

```bash
obsidian write "00-收件箱/素材/note.md" "# 我的笔记\n内容在这里"
```

### 追加内容

```bash
obsidian append "01-知识库/摄影/tips.md" "\n## 新技巧\n..."
```

### 读取内容

```bash
obsidian read "00-收件箱/note.md"
```

```

├── 00-收件箱\        ← 临时笔记

├── 04-存档\          ← 归档内容

    └── 01-内容生产\
        ├── 选题管理\
        └── 内容素材库\
```

---

## 注意事项

1. 文件名用相对路径（相对于笔记库根目录）
2. 内容用 Markdown 格式
3. 写入会覆盖已有文件，追加不会
4. 不存在的目录会自动创建
