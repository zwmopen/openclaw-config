# 主动代理技能

让 OpenClaw 从"被动响应"变成"主动行为"。

---

## 功能

### 1. 定时主动问候
- 每天早上 7:00-10:00 新会话时主动问候
- 读取简单任务表，列出当天任务

### 2. 定时提醒
- 晚上 23:00 提醒清空滴答清单
- 自定义提醒时间

### 3. 上下文感知
- 检测用户情绪 → 调整语气
- 检测对话主题 → 主动建议
- 检测时间 → 主动提醒

### 4. 智能触发
- 用户说"添加任务" → 更新任务表
- 用户说"完成任务" → 移到已完成
- 用户说"记录想法" → 保存到笔记

---

## 配置

### 心跳频率
```json
{
  "heartbeat": "5m"
}
```

### 提醒时间
```json
{
  "reminders": {
    "morning": "08:00",
    "evening": "23:00",
    "tasks": ["清空滴答清单", "检查待办事项"]
  }
}
```

---

## 使用方法

### 1. 查看任务
- 说："今天任务" / "待办"
- 自动读取简单任务表

### 2. 添加任务
- 说："添加任务：xxx"
- 自动更新简单任务表

### 3. 完成任务
- 说："完成任务：xxx"
- 自动移到已完成区域

### 4. 记录想法
- 说："记录想法：xxx"
- 自动保存到 flomo 闪念笔记

---

## 心跳检查内容

每次心跳（5分钟）检查：

1. **是否新会话 + 上午时间？**
   - 是 → 读取任务表，主动问候

2. **是否 23:00？**
   - 是 → 提醒清空滴答清单

3. **是否有用户指令？**
   - 是 → 执行对应动作

## 最佳实践

1. **心跳频率不要太快**
   - 5分钟平衡响应和性能
   - 太快会消耗更多 token

2. **提醒要有价值**
   - 不要太频繁打扰用户
   - 选择用户需要的时间点

3. **主动要有上下文**
   - 不要无缘无故主动
   - 基于用户行为和时间
