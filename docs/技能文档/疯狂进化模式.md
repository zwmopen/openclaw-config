## 工作流程

### 1. 并行任务启动
```
启动疯狂进化模式
    ↓
识别当前进化阶段
    ↓
启动多个并行任务：
  - 素材库扩充
  - 风格学习
  - 系统优化
  - 自动化开发
    ↓
每半小时汇报进度
```

### 2. 汇报机制
- **时间**：每半小时整点（如12:00, 12:30, 13:00...）
- **内容**：
  - 进化进度总览
  - 本时段完成的工作
  - 下时段计划
  - 发现的问题
- **格式**：简洁明了，数据驱动

### 3. 任务优先级

| 级别 | 名称 | 说明 | 处理方式 |
|------|------|------|----------|
| P0 | 紧急 | 立即处理 | 插队执行 |
| P1 | 高优先 | 当天处理 | 优先执行 |
| P2 | 中优先 | 3天内处理 | 正常执行 |
| P3 | 低优先 | 7天内处理 | 空闲执行 |
| P4 | 后台 | 后台持续 | 不阻塞 |
