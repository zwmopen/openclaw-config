# 技能-找技能

> 帮用户发现和安装 OpenClaw 技能

---

## 什么时候用这个技能

用户问：
- "怎么实现 X 功能？"
- "有没有 X 的技能？"
- "你能做 X 吗？"
- "我想扩展功能"
- "有没有工具/模板/工作流"

---

## 技能 CLI 是什么

**Skills CLI** (`npx skills`) 是技能包管理器。

**常用命令：**
```bash
npx skills find [关键词]    # 搜索技能
npx skills add <包名>       # 安装技能
npx skills check           # 检查更新
npx skills update          # 更新所有技能
```

**浏览技能：** https://skills.sh/

---

## 如何帮用户找技能

### 1. 理解需求

识别：
- 领域（如 React、测试、设计、部署）
- 具体任务（如写测试、创建动画、PR 审查）
- 是否有现成技能

### 2. 搜索技能

```bash
npx skills find [关键词]
```

**例子：**
- "怎么优化 React 性能？" → `npx skills find react performance`
- "帮我审查 PR" → `npx skills find pr review`
- "创建 changelog" → `npx skills find changelog`

### 3. 展示选项

告诉用户：
1. 技能名称和功能
2. 安装命令
3. 学习链接

**示例回复：**
```
找到一个技能！"vercel-react-best-practices" 提供 React 和 Next.js 性能优化指南。

安装命令：
npx skills add vercel-labs/agent-skills@vercel-react-best-practices

了解更多：https://skills.sh/vercel-labs/agent-skills/vercel-react-best-practices
```

### 4. 安装技能

```bash
npx skills add <owner/repo@skill> -g -y
```

- `-g`：全局安装
- `-y`：跳过确认

## 常见技能分类

| 分类 | 搜索关键词 |
|------|-----------|

---

## 搜索技巧

1. **用具体关键词**：搜索 "react testing" 比 "testing" 更准确
2. **尝试不同词**：搜不到 "deploy" 试试 "deployment" 或 "ci-cd"
3. **热门来源**：很多技能来自 `vercel-labs/agent-skills` 或 `ComposioHQ/awesome-claude-skills`

---

## 没找到技能怎么办

1. 告诉用户没找到
2. 用通用能力帮用户解决
3. 建议用户自己创建技能：`npx skills init my-skill`
