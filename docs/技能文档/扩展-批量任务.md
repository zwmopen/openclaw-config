# Batch Task Processor

This skill provides a systematic approach for processing large-scale tasks with progress tracking and reporting.

## When to Use

**Invoke this skill when:**
- User has a large number of files or tasks to process
- User needs systematic batch processing
- User wants progress tracking and reporting
- User mentions "batch processing", "large-scale tasks", "many files", "process all"

## Core Principles

When facing large-scale tasks, you MUST:

1. **Analyze Task Scale** - Count total tasks, processed tasks, unprocessed tasks
2. **Create Batch Plan** - Divide tasks into reasonable batches (about 50 tasks per batch)
3. **Create Todo List** - Use TodoWrite tool to create batch processing plan
4. **Report Progress in Real-time** - Tell user progress during processing
5. **Provide Final Statistics** - Give user statistics after completion

## Processing Flow

```
1. Analysis Phase
   - Count total files/tasks
   - Count processed files/tasks
   - Calculate unprocessed files/tasks
   - Report statistics to user

2. Planning Phase
   - Divide unprocessed tasks into batches (about 50 per batch)
   - Use TodoWrite to create batch processing plan
   - Show plan to user

3. Execution Phase
   - Process tasks by batch
   - Update Todo status after each batch
   - Report progress to user in real-time

4. Completion Phase
   - Count processing results
   - Report final results to user
```

## Progress Report Format

```
📊 Progress Report
- Total Tasks: XXX
- Processed: XXX
- Unprocessed: XXX
- Current Progress: XX%
- Current Batch: X/Y
```

## Key Points to Remember

- Do NOT process all tasks at once, process in batches
- Update Todo status after each batch completion
- Report progress to user in real-time
- Provide final statistics after completion
- Use Task tool for batch processing when possible

## Example Usage

```
User: Process all markdown files in this directory

Assistant:
1. Count total files: 630
2. Count processed files: 216
3. Calculate unprocessed: 414
4. Create 8 batches (about 50 files each)
5. Use TodoWrite to create plan
6. Process batch by batch
7. Report progress after each batch
8. Provide final statistics
```

## Tools to Use

- **RunCommand**: For counting files and getting file lists
- **TodoWrite**: For creating and tracking batch processing plan
- **Task**: For batch processing files efficiently
- **Write**: For creating processed files
