# 复刻写作风格

让 OpenClaw 深度学习并复刻你的写作风格。

---

## 功能

- 深度分析写作风格（5个维度）
- 竞品对比分析
- 构建选题矩阵
- 提炼标题模板库
- 持续迭代优化

---

## 使用步骤

### 1. 收集发布文章

收集 10 篇已发布文章：
- 下载为 PDF
- 或通过链接 + Relay 插件抓取
- 数量适中，保证分析效率和风格稳定性

### 2. 拆解写作风格

OpenClaw 从 5 个维度深度分析：

| 维度 | 分析内容 |
|------|---------|
| 标题公式 | 标题模式、关键词、数字用法 |
| 开头模板 | 开头方式、引入技巧 |
| 文章结构 | 段落组织、逻辑顺序 |
| 语言特点 | 口语化程度、用词习惯、句式特点 |
| 内容组织 | 信息密度、案例使用、总结方式 |

### 3. 竞品分析

找对标账号，让 OpenClaw 对比分析：

**示例：**
- 作者风格：东北老铁 + 保姆级教程
- 竞品风格：高冷极客 + 行业观察
- 建议：保留接地气特色 + 增加深度思考

### 4. 构建选题矩阵

分析现有选题比例，建议调整：

| 内容类型 | 原比例 | 建议比例 |
|---------|--------|---------|
| 教程 | 90% | 40% |
| 评测 | 0% | 20% |
| 观察 | 0% | 20% |
| 实践 | 10% | 10% |
| 避雷 | 0% | 10% |

### 5. 提炼标题模板库

在原有模板基础上，补充新模板：

**示例：**
- 原模板：「保姆级教程：xxx」
- 新模板：「xxx 最全攻略，建议收藏」
- 新模板：「一文讲透 xxx」

### 6. 整合成 SKILL.md 文件

将所有分析整合到一个文件：

- 使用场景
- 工作流程
- 核心原则
- 标准模板
- 速查表

### 7. 持续迭代

根据实际反馈优化：

**示例：**
- 问题：过度使用并列列表
- 解决：增加「去 AI 味」规则
- 规则：避免连续列表，用段落展开 + 个人感受

---

## 效果

- 写稿时间：3-4 小时 → 30 分钟
- 输出质量：风格逼真，难以分辨

## 提示词模板

### 分析写作风格

```
请分析以下 10 篇文章的写作风格：

[文章1内容]
[文章2内容]
...

从以下维度分析：
1. 标题公式
2. 开头模板
3. 文章结构
4. 语言特点
5. 内容组织

输出一份完整的写作风格指南。
```

### 竞品对比

```
请对比分析我的写作风格和以下竞品账号：

我的风格：[风格描述]
竞品风格：[竞品账号链接或描述]

给出保留什么、改进什么的建议。
```

### 生成内容

```
根据我的写作风格指南，请写一篇关于 [主题] 的文章。

要求：
- 使用我的标题模板
- 开头使用我的开头模板
- 文章结构按照我的风格
- 语言特点要像我的风格
```
