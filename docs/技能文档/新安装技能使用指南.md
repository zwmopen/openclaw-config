## 已安装技能列表

### 1. multi-search-engine-2-0-1 ⭐ 推荐
**功能**：多引擎搜索，支持17个搜索引擎，无需API Key
**触发词**：搜索、web_fetch

**支持的搜索引擎**：
- 国内（8个）：百度、Bing中文、Bing国际、360、搜狗、微信、头条、集思录
- 国外（9个）：Google、Google HK、DuckDuckGo、Yahoo、Startpage、Brave、Ecosia、Qwant、WolframAlpha

**使用示例**：
```
# 基本搜索
web_fetch({"url": "https://www.google.com/search?q=python+tutorial"})

# 站内搜索
web_fetch({"url": "https://www.google.com/search?q=site:github.com+react"})

# 文件类型搜索
web_fetch({"url": "https://www.google.com/search?q=machine+learning+filetype:pdf"})

# 时间过滤（过去一周）
web_fetch({"url": "https://www.google.com/search?q=ai+news&tbs=qdr:w"})

# 隐私搜索
web_fetch({"url": "https://duckduckgo.com/html/?q=privacy+tools"})

# 知识计算
web_fetch({"url": "https://www.wolframalpha.com/input?i=100+USD+to+CNY"})
```

### 2. self-improving ⭐ 核心
**功能**：自我进化代理，记录用户偏好、自我反思、持续学习
**触发词**：用户纠正、自我反思、偏好记录

**核心功能**：
- 自动记录用户纠正
- 自我反思并记录改进
- 分层存储（HOT/WARM/COLD）
- 自动升级/降级模式

**存储位置**：`~/self-improving/`
- `memory.md` - HOT层（≤100行，始终加载）
- `corrections.md` - 最近50条纠正
- `projects/` - 项目级学习
- `domains/` - 领域级学习
- `archive/` - 冷存储

**使用示例**：
- 用户说"不对，应该是..."→ 自动记录到corrections.md
- 完成任务后自动反思→ 记录改进建议
- 用户说"显示我的偏好"→ 显示memory.md

### 3. skill-vetter 🔒 安全
**功能**：技能安全审查，安装前检查安全风险
**触发词**：审查技能、安装前检查

**检查项目**：
- 源代码审查
- 权限范围检查
- 红旗警告（外部请求、凭证访问、系统文件修改等）
- 风险分级（LOW/MEDIUM/HIGH/EXTREME）

**红旗警告**：
```
🚨 立即拒绝：
- curl/wget到未知URL
- 发送数据到外部服务器
- 请求凭证/token/API key
- 读取~/.ssh, ~/.aws等敏感目录
- 访问MEMORY.md, USER.md等
- 使用base64解码
- 使用eval()或exec()
- 修改系统文件
- 安装未列出的包
- 访问浏览器cookie
```

### 4. openclaw-tavily-search
**功能**：Tavily搜索API
**需要**：TAVILY_API_KEY环境变量

**使用示例**：
```bash
# 基本搜索
python3 scripts/tavily_search.py --query "..." --max-results 5

# 包含简短答案
python3 scripts/tavily_search.py --query "..." --max-results 5 --include-answer

# Markdown格式
python3 scripts/tavily_search.py --query "..." --max-results 5 --format md
```

### 6. qmd
**功能**：本地文件搜索和索引（BM25 + 向量 + 重排序）
**需要**：安装qmd CLI

**使用示例**：
```bash
# 添加集合
qmd collection add /path --name docs --mask "**/*.md"

# 更新索引
qmd update

# 搜索
qmd search "query"      # BM25
qmd vsearch "query"     # 向量
qmd query "query"       # 混合

# 获取文档
qmd get docs/path.md:10 -l 40
```
