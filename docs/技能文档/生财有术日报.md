# 生财有术日报 - 技能定义

## 触发词
- 生财日报
- 抓取生财有术
- scys daily

## 功能
每天自动抓取生财有术热门帖子，生成日报推送到飞书群

## 工作流程

### 1. 打开网站
```
browser action=open targetUrl=https://scys.com profile=openclaw
```

### 2. 等待登录（如果需要）
- 检查是否已登录
- 如果未登录，提示用户扫码
- 登录状态会保存在浏览器中

### 3. 抓取热门帖子
- 点击"热门"标签
- 等待页面加载
- 提取帖子列表：
  - 标题
  - 作者
  - 点赞数
  - 评论数
  - 标签
  - 链接
  - AI速读摘要（如果有）

### 4. 生成日报格式
```markdown
## 📰 生财有术日报 - {日期}

### 🔥 今日热门

**1. {标题}**
👤 {作者} | 👍{点赞} 💬{评论} | 🏷️ {标签}
🔗 链接：{链接}
📝 {AI速读/核心内容}

**2. {标题}**
...

---
💡 由 OpenClaw 自动抓取推送
```

### 5. 推送到飞书
- 使用 message 工具发送到指定群

## 定时任务配置

### 日报（每天早上9点）
- 时间：每天 09:00
- 内容：热门帖子前10条
- 推送：飞书群

### 周报（每周一早上9点）
- 时间：每周一 09:00
- 内容：精华帖子前15条
- 推送：飞书群

## 技术实现

### 浏览器自动化
- 使用 OpenClaw 内置浏览器
- 登录状态持久化
- 支持无头模式（后台运行）

### 数据提取
- 使用 browser snapshot 获取页面结构
- 解析帖子列表
- 提取关键信息

### 错误处理
- 登录失效时通知用户
- 网络超时重试
- 保存上次成功抓取的数据

## 配置项

```yaml
scys_daily:
  enabled: true
  schedule:
    daily: "09:00"
    weekly: "Monday 09:00"
  push:
    channel: feishu
    chat_id: "oc_e348903727eebe22642cf3cbbe400f99"
  filters:
    - tags: [AI, 小红书, 自媒体]  # 可选：只推送特定标签
    - min_likes: 30               # 可选：最低点赞数
  count:
    daily: 10
    weekly: 15
```

## 注意事项

1. **登录状态**：首次使用需要用户扫码登录
2. **频率限制**：每天最多抓取3次，避免对服务器造成压力
3. **内容筛选**：可以配置只推送特定标签的内容
4. **数据保存**：抓取的数据保存到本地，方便后续分析
