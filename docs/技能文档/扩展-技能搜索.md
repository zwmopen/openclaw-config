# 技能搜索器

## 核心功能

当用户提出需求时，主动搜索并集成相关技能，不断扩展AI能力边界。

## 触发条件

- 用户提出新类型任务
- 用户说"我想..."、"帮我..."、"能不能..."
- 现有技能无法满足需求
- 用户明确要求搜索技能

## 搜索来源

### 1. ClawHub 技能库（小龙虾）⭐ 推荐
- **网址**: https://clawhub.ai/skills
- **特点**: 中文AI技能库，下载量排序
- **分类**: 办公、开发、生活、学习
- **格式**: 可直接导入使用

### 2. Claude 技能仓库
- **GitHub**: https://github.com/anthropics/anthropic-cookbook
- **内容**: Claude提示词最佳实践
- **特点**: 官方维护，质量高

### 3. OpenClaw 中文技能库
- **GitHub**: awesome-openclaw-skills-zh
- **特点**: 几百款实用技能，中文友好
- **分类**: 办公、开发、日常

### 4. GitHub 技能仓库
- **awesome-prompts**: 提示词集合
- **awesome-ai-agents**: AI智能体
- **prompt-engineering**: 提示工程

### 5. 其他平台
- **OpenAI GPTs**: https://chat.openai.com/gpts
- **LangChain Hub**: https://smith.langchain.com/hub
- **扣子 Coze**: https://www.coze.cn

## 搜索流程

```
1. 分析用户需求
   ↓
2. 提取关键词
   ↓
3. 优先搜索 ClawHub（中文友好）
   ↓
4. 搜索 GitHub 仓库
   ↓
5. 筛选高质量技能
   ↓
6. 改编为 Trae 技能格式
   ↓
7. 保存到技能库
   ↓
8. 更新记忆文件
```

## 技能格式转换

### ClawHub 格式 → Trae 格式
```
ClawHub:
---
name: "技能名"
description: "描述"
---
# 内容

Trae:
---
name: "技能名"
description: "描述。触发条件。"
---
# 内容
```

## 集成位置

| 类型 | 位置 | 说明 |
|------|------|------|
| 全局技能 | `C:\Users\z\.trae-cn\skills\` | 所有项目可用 |
| 项目技能 | `项目\.trae\skills\` | 仅当前项目 |
| 记忆文件 | `项目\.trae\memory\` | 用户偏好和历史 |

## 常用技能关键词

### 动作类
- 批量处理、格式转换、内容生成
- 数据分析、文件整理、自动化

### 对象类
- 文件、图片、视频、文本
- Excel、PDF、Markdown

### 目标类
- 压缩、整理、格式化、优化
- 翻译、总结、提取

## 使用示例

**用户**: "帮我批量重命名文件"

**AI 行动**:
1. 搜索 ClawHub: `file rename`
2. 搜索 GitHub: `batch rename skill`
3. 找到最佳方案
4. 创建 `file-renamer` 技能
5. 保存到 `C:\Users\z\.trae-cn\skills\`
6. 更新记忆文件
7. 执行任务

## 自我改进

每次搜索和集成技能后：
1. 记录技能来源
2. 评估技能质量
3. 优化技能格式
4. 更新搜索策略
