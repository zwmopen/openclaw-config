## ⚙️ 配置参数

### 高频模式（需申请无限额度）

| 参数 | 值 | 说明 |
|------|-----|------|
| 心跳间隔 | 35秒 | 高频检查 |
| 消息频率 | 实时 | 每次思考都发消息 |
| 定时提醒 | 全部开启 | 眼睛、喝水、番茄钟等 |
| 进化汇报 | 每小时 | 详细进度 |

### 低频模式（默认，节省API）

| 参数 | 值 | 说明 |
|------|-----|------|
| 心跳间隔 | 60分钟 | 低频检查 |
| 消息频率 | 每小时 | 每小时汇报一次 |
| 定时提醒 | 只保留早间+复盘 | 节省消息 |
| 进化汇报 | 记录到文件 | 不发消息 |

## ⚠️ 注意事项

### 飞书API限制

**限制**：每月1万条消息
**高频模式预估**：~450条/月（基础提醒）
**低频模式预估**：~60条/月

**切换条件**：
1. 飞书API申请无限额度成功
2. 用户明确要求切换

**配置备份**：
- 高频配置：`docs/高频发消息配置备份.md`
- 临时配置：`HEARTBEAT.md`（当前生效）

## 🔧 技术实现

### 心跳机制

```javascript
// 高频模式
setInterval(() => {
  // 检查定时任务
  // 发送消息（如果需要）
}, 35 * 1000);

// 低频模式
setInterval(() => {
  // 检查定时任务
  // 只在整点发送消息
}, 60 * 60 * 1000);
```

### 消息发送逻辑

```javascript
if (isHighFrequency) {
  // 实时发送
  sendMessage(thought);
} else {
  // 记录到文件
  appendToFile(thought);
  // 每小时发送一次
  if (isOnTheHour) {
    sendSummary();
  }
}
```
