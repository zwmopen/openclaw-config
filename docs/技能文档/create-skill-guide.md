# 创建新技能

> 当用户让我创建技能时，按这个流程操作

---

## 技能库位置

**主技能库：** `D:\Program Files\Obsidian\zwm\.zwm\OpenClaw\`

所有技能文档都放在这里。

---

## 创建流程

### 1. 确认技能名称
用户说："创建技能 记录选题"

### 2. 创建文件
路径：`D:\Program Files\Obsidian\zwm\.zwm\OpenClaw\skill-记录选题.md`

### 3. 技能模板

```markdown
---
name: "技能名称"
description: "技能描述。Invoke when 用户说什么时调用。"
---

# 技能名称

## 触发词

| 用户说 | 动作 |
|--------|------|
| 触发词1 | 动作1 |
| 触发词2 | 动作2 |

## 功能说明

详细描述技能功能。

## 使用方法

### 输入
- 输入内容

### 输出
- 输出内容

### 示例

```
用户：触发词
我：执行动作
```

## 注意事项

- 注意事项1
- 注意事项2

---

创建时间：YYYY-MM-DD
```

### 4. 更新索引
在 `skill-catalog.md` 中添加新技能。

### 5. 更新触发规则
在 `HEARTBEAT.md` 的技能快速识别表中添加触发词。
