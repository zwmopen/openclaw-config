## 错误处理

### 遇到错误时

1. **检查 `SELF-HEALING.md`** - 查看自愈策略
2. **尝试自动修复** - 按重试机制执行
3. **记录错误日志** - 更新错误统计
4. **通知用户** - 如需用户介入

### 常见错误

| 错误 | 处理方式 |
|------|----------|
| API 限流 | 等待 60 秒后重试 |
| 超时 | 重试 3 次 |
| 文件不存在 | 创建文件 |
| 进程阻塞 | 后台运行 |
