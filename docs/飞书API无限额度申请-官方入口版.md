## ⚠️ 常见问题

### Q1: 找不到"API 配额"入口？

**解决方法**：
1. 确认你已经登录飞书开放平台
2. 确认你已经进入应用（点击应用名称）
3. 左侧菜单往下滚动，找到"API 配额"
4. 如果还是找不到，用搜索框搜索"API 配额"

### Q2: 找不到"申请提升配额"按钮？

**解决方法**：
1. 确认你的应用是"自建应用"（不是企业应用）
2. API配额页面应该有"申请提升配额"按钮
3. 如果没有，可能是权限问题，联系飞书客服

### Q3: 表单无法提交？

**解决方法**：
1. 使用官方入口（https://open.feishu.cn/）
2. 不要用浏览器打开表单
3. 在飞书App内打开表单（如果需要）
