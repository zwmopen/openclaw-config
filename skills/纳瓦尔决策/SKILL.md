# 纳瓦尔财富与清醒操作系统

## 快速触发

```
纳瓦尔：[你的问题或困境]
```

## 核心框架

### 财富创造公式

```
财富 = 专属知识 × 杠杆 × 判断力 × 责任感
```

### 杠杆优先级

1. 代码（最高）：一次创作，无限售卖
2. 媒体（次之）：一次创作，长期复利
3. 资本：钱自动工作
4. 劳动力（谨慎）：最难规模化

### 决策协议

1. 最小化后悔：80岁时会后悔没做吗？
2. 可逆性测试：能撤销吗？
3. 收益风险比：至少3:1
4. 杠杆倍增：只做提升杠杆的事
5. 选择权检查：选择创造更多选项的路

### 思考框架

1. 第一性原理检查：本质上什么是真的？
2. 动机分析：所有参与者的真实诉求
3. 二阶思维：然后会发生什么？
4. 选择权评估：会消耗多少选择权？
5. 非对称回报筛选：收益是风险的10倍以上吗？
6. 专属知识核查：能被培训或外包吗？
7. 杠杆识别：离开我还能自动运转吗？
8. 长期游戏测试：未来10年还愿意做吗？

## 适用场景

- 职业选择
- 商业决策
- 财富方向
- 长期规划
- 杠杆选择
- 决策困境
