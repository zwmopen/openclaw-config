# 去AI味写作技能

## 核心理念

AI输出的内容有一股机器味，通过禁用词库、精准受众、非线性表达，可以让AI输出的内容更像真人。

---

## 执行规则

### 第一步：设置精准受众

**错误写法**：
```
面向职场新人
```

**正确写法**：
```
给那个每天下午三点还没写完周报、想早点下班去健身的运营小哥
```

**受众画像四要素**：
1. 具体的人（谁）
2. 具体的烦恼（痛点）
3. 讨厌什么（避免）
4. 只关心什么（核心）

---

### 第二步：应用禁用词库

**强制禁用词**：
```
总之、不仅如此、关键在于、深入探讨、开启新篇章、更重要的是、值得注意的是、是一个强有力的工具、解锁潜力、极具吸引力的、令人惊叹的、全方位的、多维度的。
```

**禁用标点**：
```
""双引号、破折号——
```

**禁用结构**：
```
首先、其次、最后、第一、第二、第三
```

---

### 第三步：强制非线性表达

**禁用**：
- 有序列表（1. 2. 3.）
- 排比句
- 对仗句
- 总分总结构

**使用**：
- 短句
- 口语化转折（说白了、其实没那么玄乎、你想啊）
- 自然过渡
- 戛然而止

---

### 第四步：替换形容词

**原则**：把形容词替换成具体的动词或事实

| 机器味 | 人味 |
|--------|------|
| 重要的是... | 直接影响了XX% |
| 非常有效... | 用了之后XX提升了3倍 |
| 极具吸引力... | 100个人里有80个点了进去 |

---

### 第五步：最后调整

**调整指令**：
```
现在的内容太像机器人写的了，请删掉所有无意义的铺垫和总结，把操作步骤再细化三倍，加入具体的案例或失败经历。
```

## 完整提示词模板

```
请在写作时严格执行以下规则：

1. 严禁使用任何宏大的开场白，直接进入主题。
2. 删掉所有总结性陈词。
3. 把所有的形容词替换成具体的动词或事实。
4. 禁止使用排比句和过于规整的对仗。

你的读者是：[具体的人+具体的烦恼+讨厌什么+只关心什么]

严禁使用：总之、不仅如此、关键在于、深入探讨、开启新篇章、更重要的是、值得注意的是、是一个强有力的工具、解锁潜力、极具吸引力的、令人惊叹的、全方位的、多维度的。

禁止使用""双引号、破折号——。

禁止使用有序列表，多用短句，加入口语化转折，比如说白了、其实没那么玄乎。

主题：[你要写的主题]
```

## 使用示例

### 示例1：小红书笔记

**触发**：
```
去AI味：如何利用下班时间做副业

受众：在杭州工作三年、月薪一万五但觉得没什么存款的运营小哥，他讨厌听大道理，只关心怎么在下班后搞点副业，每个月多赚个三五千。
```

**输出**：
```
你每天下班刷手机刷到十一点，最后什么也没落下。

说白了，就是闲的。

想搞副业？先别想那么多，从最简单的开始。杭州这边的运营小哥，下班跑个滴滴，一个月也能多赚两三千。

其实没那么玄乎，就是干不干的问题。
```

---

### 示例2：公众号文章

**触发**：
```
去AI味：如何选择靠谱的化妆培训机构

受众：在杭州工作、想学化妆但不知道哪家培训机构靠谱的女生，她担心被坑钱、担心学不会、担心找不到工作。
```

**输出**：
```
想学化妆，但怕被坑？

你纠结的不是学不学，而是去哪学。

杭州这边化妆培训学校不少，但靠谱的就那几家。我见过太多人，报了名才发现老师根本不教实用的，全是理论。

说白了，选学校就三点：看老师、看作品、看就业。

老师要是还在接单的，作品要是最近拍的，就业要是能查到真人的。

别听那些包就业的鬼话，骗人的。
```

## 快速触发词

| 触发词 | 功能 |
|--------|------|
| `去AI味：[主题]` | 使用去机器味模板生成内容 |
| `真人味：[主题]` | 使用口语化风格生成内容 |
| `接地气：[主题]` | 使用具体受众生成内容 |
| `太机器了` | 删掉所有禁用词，加入口语化转折 |
| `太理论了` | 加入具体案例和失败经历 |
| `太规整了` | 打破有序列表，用短句和转折 |

---

## 注意事项

1. **不要指望第一遍就完美**：AI需要多次调整
2. **针对性压榨**：告诉AI具体哪里不好
3. **保持真实感**：不要太刻意追求口语化
4. **具体化**：受众越具体，效果越好
