# 自动化模式技能

> 通过系统命令和脚本控制电脑，实现一键启动各种模式

---

## 🎵 音乐模式

**触发词**：`音乐模式`、`播放音乐`、`听歌`

**功能**：
1. 调节音量到50%
2. 打开网易云音乐
3. 等待20秒加载
4. 发送空格键播放

**脚本位置**：
- PowerShell: `D:\AI编程\openclaw\scripts\music-mode.ps1`
- AutoHotkey: `D:\AI编程\openclaw\scripts\music-mode.ahk`

**使用方式**：
- 用户说"音乐模式"或"播放音乐"
- AI执行PowerShell脚本
- 或用户按 `Ctrl+Alt+M`（需要先运行AHK脚本）

---

## 💼 工作模式

**触发词**：`工作模式`、`开始工作`

**功能**：
1. 调节亮度到70%

3. 打开Obsidian
4. 提示打开任务清单

- PowerShell: `D:\AI编程\openclaw\scripts\work-mode.ps1`

**使用方式**：
- 用户说"工作模式"或"开始工作"
- AI执行PowerShell脚本
