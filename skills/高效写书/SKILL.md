# 高效写书技能

## 快速触发词

| 触发词 | 功能 |
|--------|------|
| `出书概念：[领域]` | 生成5个书籍概念 |
| `出书大纲：[书名]` | 生成10-15章大纲 |
| `出书正文：[章节]` | 生成章节初稿 |
| `出书故事：[概念]` | 生成8个原创故事 |
| `出书证据：[主题]` | 搜集权威证据 |

---

## 核心流程

```
概念验证 → 大纲构建 → 正文撰写 → 故事填充 → 增强说服力
```

---

## 提示词1：概念验证

**使用**：`出书概念：[领域]`

```
假设你是一位资深的出版策略师。请为【领域】领域开发五个具有商业可行性的书籍概念。

每个概念需详细包括：
1. 强有力的书名
2. 清晰的目标读者画像
3. 差异化定位（区别于市场上现有书籍）
4. 市场规模评估
5. 读者愿意花钱购买的核心原因
6. 相关的最新行业趋势

请帮助我在投入数月心血前，有效验证这些高潜力创意。
```

---

## 提示词2：大纲构建

**使用**：`出书大纲：[书名]`

```
请为目标受众构建《【书名】》这本书的逐章大纲框架。全书请包含 10-15 章。

每一章的大纲需要详细列出：
1. 以读者效益为导向的章节标题
2. 本章核心概念
3. 预计字数
4. 读者读完本章后的认知或行为转变
5. 开篇的吸引力"钩子"（Hook）
6. 结尾的总结与悬念
7. 向下一章的自然过渡提示
```

---

## 提示词3：正文撰写

**使用**：`出书正文：第X章《标题》主题`

```
请为本书的第【章号】章《【章节标题】》起草完整的手稿初稿，本章将专注于探讨【核心主题】。

内容必须按顺序包括：
• 引人入胜的开场白
• 3-4 个带有清晰小标题的核心段落
• 具体的现实示例或案例研究
• 具有实操性的见解/建议
• 自然过渡到下一章的结尾

要求：通篇使用强有力、积极主动且引人入胜的语言风格。
```

---

## 提示词4：故事填充

**使用**：`出书故事：[概念]`

```
请为我的书创作 8 个原创的叙事片段或小故事，用于生动演示书中的核心概念。

每个故事字数控制在 150-250 字之间，并严格满足以下要求：
• 包含生动的细节描写和真实的对话
• 有清晰完整的叙事弧线（起承转合）
• 能够与我的目标受众产生强烈共鸣
• 读完后能提供有意义的启发与收获，而不是生硬的说教。
```

---

## 提示词5：增强说服力

**使用**：`出书证据：[主题]`

```
针对第【章号】章关于【具体主题】的内容，请帮我汇编并整合相关的权威研究与证据。

具体需要包含以下维度的内容：
• 3-5 篇相关的同行评审研究成果或学术论文
• 2-3 个来自可信数据源、引人注目的统计数据
• 1-2 位该领域公认权威专家的经典引言
• 1 个高度相关的商业/现实案例研究

请务必在末尾包含正确的参考文献或引用来源说明。
```

## 去AI味调整

**在提示词3之后**，使用去AI味规则：

```
现在的内容太像机器人写的了，请删掉所有无意义的铺垫和总结，把操作步骤再细化三倍，加入具体的案例或失败经历。

严禁使用：总之、不仅如此、关键在于、深入探讨、开启新篇章、更重要的是、值得注意的是、是一个强有力的工具、解锁潜力、极具吸引力的、令人惊叹的、全方位的、多维度的。

禁止使用有序列表，多用短句，加入口语化转折，比如说白了、其实没那么玄乎。
```

## 进阶建议

### 1. 迭代优化

- 生成初稿后，继续让AI"扩写"、"润色"、"改变语气"
- 不满意的部分，单独拎出来重新生成

### 2. 注入个人风格

- 加入真实经历
- 加入独家案例
- 加入个人观点

### 3. 避免AI味

- 使用去AI味规则
- 加入口语化转折
- 打破有序列表

## 注意事项

1. **按顺序使用**：从概念验证开始，逐步推进
2. **替换占位符**：将【】中的内容替换为实际信息
3. **迭代优化**：一次不完美，多轮润色
4. **注入个人风格**：加入真实经历和独家案例
5. **避免AI味**：使用去AI味写作技巧
