# 智能建议与洞察报告技能

## 功能概述

本技能实现两大核心功能：
1. **智能建议** - 每日复盘时根据历史数据给出改进建议
2. **洞察报告** - 每周/每月自动生成深度分析报告

---

## 一、智能建议功能

### 触发时间
每天晚上 21:00（深度复盘时自动执行）

### 数据来源
- 情绪日记：`个人成长/情绪日记/YYYY-MM-DD.md`
- 财务记录：`个人成长/财务管理/YYYY-MM-DD.md`
- 任务完成：`00待办事项/📅 今日任务.md`
- 健康数据：睡眠、运动、喝水记录

### 建议类型

#### 1. 情绪优化建议
**触发条件**：
- 情绪评分 < 5分
- 情绪波动 > 3分
- 连续2天情绪低分

**建议内容**：
- 识别情绪触发因素
- 提供情绪调节方法
- 建议休息或调整计划

**示例**：
```
📊 情绪分析：
- 今日情绪评分：4分（偏低）
- 触发因素：任务过多，压力过大
- 建议：
  1. 明天减少任务量，优先完成核心任务
  2. 安排30分钟休息时间
  3. 晚上21:00前完成所有工作，避免熬夜
```

#### 2. 效率提升建议
**触发条件**：
- 任务完成率 < 60%
- 深度工作时间 < 4小时
- 频繁切换任务

**建议内容**：
- 识别低效时段
- 优化时间分配
- 建议时间盒规划

**示例**：
```
📊 效率分析：
- 今日任务完成率：50%（偏低）
- 高效时段：9:00-11:00, 14:00-16:00
- 低效时段：13:00-14:00（饭后犯困）
- 建议：
  1. 重要任务安排在高效时段
  2. 午休调整为12:30-13:30
  3. 减少13:00-14:00的任务安排
```

#### 3. 习惯养成建议
**触发条件**：
- 习惯完成率 < 80%
- 连续2天未完成习惯
- 新习惯建立初期

**建议内容**：
- 分析中断原因
- 提供习惯绑定建议
- 降低习惯难度

**示例**：
```
📊 习惯分析：
- 今日习惯完成率：60%（偏低）
- 未完成习惯：晨间宣言、俯卧撑
- 中断原因：起床时间推迟
- 建议：
  1. 晨间宣言绑定到刷牙后
  2. 俯卧撑调整为起床后立即做
  3. 设定7:00起床闹钟
```

#### 4. 财务优化建议
**触发条件**：
- 单日支出 > 200元
- 不必要支出 > 50元
- 月初支出过快

**建议内容**：
- 识别不必要支出
- 提供预算优化建议
- 建议支出分类

**示例**：
```
📊 财务分析：
- 今日支出：280元（偏高）
- 不必要支出：奶茶30元、外卖50元
- 建议：
  1. 明天减少外卖，自己做饭
  2. 用白开水代替奶茶
  3. 剩余预算：本月还剩22天，建议每天支出控制在100元内
```

#### 5. 健康改善建议
**触发条件**：
- 睡眠评分 < 5分
- 运动未达标
- 连续3天熬夜

**建议内容**：
- 分析睡眠质量
- 提供运动建议
- 建议生活方式调整

**示例**：
```
📊 健康分析：
- 昨晚睡眠：5小时（不足）
- 今日运动：俯卧撑60个 ✅
- 建议：
  1. 今晚23:00前入睡，保证7小时睡眠
  2. 明天增加跑步30分钟
  3. 减少咖啡因摄入
```

## 二、周洞察报告

### 触发时间
每周日晚上 21:00

### 数据范围
本周一至本周日（共7天）

### 分析维度

#### 1. 情绪趋势分析
**数据来源**：`个人成长/情绪日记/YYYY-MM-DD.md`（7天）

**分析内容**：
- 本周情绪平均值
- 情绪波动范围（最高分 - 最低分）
- 情绪触发因素TOP3
- 情绪管理效果评估

**输出示例**：
```markdown
## 😊 情绪趋势分析

### 本周情绪概况
- 平均情绪评分：6.5/10
- 情绪波动范围：4-8分（波动4分）
- 情绪稳定性：中等

### 情绪触发因素TOP3
1. 任务压力（3次）
2. 睡眠不足（2次）
3. 天气影响（1次）

### 情绪管理效果
- 使用情绪调节方法：3次
- 有效率：66%
- 建议方法：深呼吸、散步、听音乐

### 下周建议
- 避免任务堆积，提前规划
- 保证充足睡眠
- 天气不好时安排室内活动
```

#### 2. 任务完成分析
**数据来源**：`00待办事项/📅 今日任务.md`（7天）

**分析内容**：
- 本周任务完成率
- 高效时段识别
- 低效时段识别
- 任务类型分析

**输出示例**：
```markdown
## 📋 任务完成分析

### 本周任务概况
- 总任务数：35个
- 完成任务：28个
- 完成率：80%

### 高效时段TOP3
1. 9:00-11:00（完成15个任务）
2. 14:00-16:00（完成8个任务）
3. 20:00-21:00（完成5个任务）

### 低效时段TOP3
1. 13:00-14:00（完成1个任务）
2. 17:00-18:00（完成2个任务）
3. 22:00-23:00（完成2个任务）

### 任务类型分析
- 内容创作：12个（完成10个）
- 代运营工作：10个（完成9个）
- 学习提升：8个（完成6个）
- 生活事务：5个（完成3个）

### 下周建议
- 重要任务安排在9:00-11:00
- 13:00-14:00安排休息或低难度任务
- 增加内容创作时间
```

#### 3. 习惯执行分析
**数据来源**：`00待办事项/🔄 每日循环.md`（7天）

**分析内容**：
- 本周习惯完成率
- 各习惯完成情况
- 中断原因分析
- 习惯巩固建议

**输出示例**：
```markdown
## ⚡ 习惯执行分析

### 本周习惯概况
- 总习惯次数：105次（15个习惯 × 7天）
- 完成次数：84次
- 完成率：80%

### 各习惯完成情况
| 习惯 | 目标次数 | 完成次数 | 完成率 |
|------|---------|---------|--------|
| 晨间宣言 | 14 | 14 | 100% ✅ |
| 深度工作 | 14 | 12 | 86% |
| 俯卧撑 | 7 | 6 | 86% |
| 跑步 | 7 | 4 | 57% ⚠️ |
| 晚间复盘 | 7 | 7 | 100% ✅ |

### 中断原因分析
- 跑步中断原因：雨天2次、太累1次
- 深度工作中断原因：被打断2次

### 下周建议
- 跑步调整为室内运动（雨天备选方案）
- 深度工作时间关闭通知
- 保持晨间宣言和晚间复盘的100%完成率
```

#### 4. 财务分析
**数据来源**：`个人成长/财务管理/YYYY-MM-DD.md`（7天）

**分析内容**：
- 本周总支出
- 支出结构分析
- 不必要支出识别
- 预算执行情况

**输出示例**：
```markdown
## 💰 财务分析

### 本周财务概况
- 本周总支出：1,200元
- 日均支出：171元
- 本月预算：3,000元
- 已使用：1,200元（40%）

### 支出结构分析
| 类别 | 金额 | 占比 |
|------|------|------|
| 餐饮 | 500元 | 42% |
| 交通 | 200元 | 17% |
| 娱乐 | 300元 | 25% |
| 其他 | 200元 | 17% |

### 不必要支出
- 奶茶：70元
- 外卖：150元
- 游戏：50元
- 合计：270元（23%）

### 下周建议
- 减少外卖，自己做饭（可节省150元）
- 减少奶茶摄入（可节省70元）
- 控制娱乐支出在200元内
- 下周预算：800元
```

#### 5. 健康分析
**数据来源**：睡眠、运动、喝水记录（7天）

**分析内容**：
- 本周睡眠质量平均值
- 运动完成情况
- 健康风险评估
- 生活方式改善建议

**输出示例**：
```markdown
## 💪 健康分析

### 本周健康概况
- 平均睡眠时长：6.5小时
- 平均睡眠评分：6/10
- 运动完成率：75%
- 喝水达标率：85%

### 睡眠质量分析
- 最佳睡眠：周三（7.5小时，评分8/10）
- 最差睡眠：周五（5小时，评分4/10）
- 熬夜次数：2次

### 运动完成情况
- 俯卧撑：6/7天（86%）
- 跑步：4/7天（57%）
- 总运动时长：180分钟

### 健康风险评估
- 睡眠不足风险：⚠️ 中等
- 运动不足风险：⚠️ 低
- 压力风险：⚠️ 中等

### 下周建议
- 保证每天7小时睡眠
- 增加跑步频率（目标5次/周）
- 喝水量保持在8杯/天
- 23:00前入睡
```

#### 6. 下周行动计划
**综合分析**后生成：

```markdown
## 🎯 下周行动计划

### 重点目标（3个）
1. 保证每天7小时睡眠（目标：睡眠评分≥7/10）
2. 任务完成率提升到85%
3. 减少不必要支出（目标：<150元）

### 习惯养成重点（2个）
1. 跑步（目标：5次/周）
2. 深度工作（目标：每次不被打断）

### 时间分配调整
- 高效时段（9:00-11:00）：重要任务
- 低效时段（13:00-14:00）：休息或低难度任务
- 晚间时段（20:00-21:00）：复盘和规划

### 预算控制重点
- 下周预算：800元
- 餐饮控制在350元内
- 娱乐控制在150元内
```

### 输出格式

**周洞察报告**存放在：`个人成长/周报告/YYYY-WXX.md`

## 三、月洞察报告

### 触发时间
每月最后一天晚上 21:00

### 数据范围
本月第1天至最后一天

### 分析维度

#### 1. 月度总结
- 本月任务完成率
- 本月情绪平均值
- 本月财务总结
- 本月健康总结

#### 2. 趋势分析
- 情绪趋势（周对比）
- 任务完成趋势（周对比）
- 财务趋势（周对比）
- 健康趋势（周对比）

#### 3. 亮点与问题
- 本月做得好的地方
- 本月存在的问题
- 改进机会识别

#### 4. 下月计划
- 重点目标（3-5个）
- 习惯养成重点（3个）
- 时间分配策略
- 预算控制目标

### 输出格式

**月洞察报告**存放在：`个人成长/月报告/YYYY-MM.md`

## 四、实施细节

### 自动化流程

#### 每日流程（21:00）
```
1. 读取今日数据
   ├─ 情绪日记
   ├─ 财务记录
   ├─ 任务完成
   └─ 健康数据

2. 分析数据
   ├─ 识别异常
   ├─ 发现亮点
   └─ 提取洞察

3. 生成智能建议
   ├─ 情绪优化建议
   ├─ 效率提升建议
   ├─ 习惯养成建议
   ├─ 财务优化建议
   └─ 健康改善建议

4. 输出建议
   ├─ 存放到文件
   └─ 发送给用户
```

#### 每周流程（周日21:00）
```
1. 读取本周数据（7天）
   ├─ 情绪日记
   ├─ 财务记录
   ├─ 任务完成
   └─ 健康数据

2. 分析趋势
   ├─ 情绪趋势
   ├─ 任务完成趋势
   ├─ 财务趋势
   └─ 健康趋势

3. 生成周报告
   ├─ 情绪趋势分析
   ├─ 任务完成分析
   ├─ 习惯执行分析
   ├─ 财务分析
   ├─ 健康分析
   └─ 下周行动计划

4. 输出报告
   ├─ 存放到文件
   └─ 发送给用户
```

#### 每月流程（最后一天21:00）
```
1. 读取本月数据（全月）
   ├─ 情绪日记
   ├─ 财务记录
   ├─ 任务完成
   └─ 健康数据

2. 分析趋势
   ├─ 情绪趋势（周对比）
   ├─ 任务完成趋势（周对比）
   ├─ 财务趋势（周对比）
   └─ 健康趋势（周对比）

3. 生成月报告
   ├─ 月度总结
   ├─ 趋势分析
   ├─ 亮点与问题
   └─ 下月计划

4. 输出报告
   ├─ 存放到文件
   └─ 发送给用户
```
