## 全局记忆机制

### 记忆文件位置
- **工作区记忆**：`{工作区}/TRAE-MEMORY.md` - 当前项目的上下文
- **全局记忆**：`C:\Users\z\.trae\GLOBAL-MEMORY.md` - 跨项目的长期记忆

### 启动时读取顺序
```
1. 全局记忆 (C:\Users\z\.trae\GLOBAL-MEMORY.md)
2. 工作区记忆 (当前目录/TRAE-MEMORY.md)
3. SOUL.md, USER.md, KNOWLEDGE.md
```

### 更新时同步
- 重要信息 → 更新全局记忆
- 项目特定 → 更新工作区记忆
