## 错误类型

### 1. API 错误
- **类型**：API 调用失败
- **原因**：网络问题、限流、服务器错误
- **重试**：✅ 支持
- **策略**：指数退避，最大重试 3 次

### 2. 文件错误
- **类型**：文件操作失败
- **原因**：权限问题、文件不存在、路径错误
- **重试**：⚠️ 部分支持
- **策略**：检查权限，尝试替代路径

### 3. 认知错误
- **类型**：理解错误、逻辑错误
- **原因**：上下文不足、指令模糊
- **重试**：❌ 不支持
- **策略**：请求用户澄清

### 4. 系统错误
- **类型**：OpenClaw 系统错误
- **原因**：配置错误、资源不足
- **重试**：⚠️ 部分支持
- **策略**：等待系统恢复，通知用户

## 重试策略

### 指数退避算法

```python
def calculate_delay(attempt, base_delay=1, max_delay=60):
    """
    计算重试延迟（指数退避）
    
    Args:
        attempt: 当前重试次数（从1开始）
        base_delay: 基础延迟（秒）
        max_delay: 最大延迟（秒）
    
    Returns:
        延迟时间（秒）
    """
    delay = base_delay * (2 ** (attempt - 1))
    return min(delay, max_delay)

# 重试延迟示例：
# 第1次重试：1秒
# 第2次重试：2秒
# 第3次重试：4秒
# 第4次重试：8秒
# ...
```

### 最大重试次数

| 错误类型 | 最大重试 | 说明 |
|---------|---------|------|
| API 限流 | 3次 | 等待限流解除 |
| 网络错误 | 3次 | 等待网络恢复 |
| 文件权限 | 1次 | 尝试替代路径 |
| 系统错误 | 2次 | 等待系统恢复 |
